# Нейросети: основы

Путь от одного нейрона до обучаемой сети без чёрных ящиков: взвешенная сумма и активация, прямой проход слоёв, функции потерь, обратное распространение и градиентный спуск, переобучение и регуляризация, обзор архитектур (MLP, CNN, RNN/LSTM, Transformer) и эмбеддинги с косинусной близостью. Каждое понятие проверяется в живой Python-песочнице.

Сгенерировано через `lesson-forge`.
