# Git с нуля до уверенного

Практический курс по Git: от модели «снимков» и первого коммита до веток, слияний, разрешения конфликтов, отката изменений, работы с удалёнными репозиториями, rebase, stash, cherry-pick, тегов, reflog и командных рабочих процессов. Каждый раздел даёт живой терминал в браузере, где команды выполняются по-настоящему.

Сгенерировано через `lesson-forge`.
