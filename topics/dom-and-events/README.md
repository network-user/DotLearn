# DOM и события в браузере

Тема о том, как JavaScript видит и меняет страницу. Семь концептов проводят от дерева DOM и выборки узлов через создание, изменение и удаление элементов, различие атрибутов и свойств, модель событий с погружением и всплытием, `addEventListener` и объект `Event`, делегирование событий - к прикладным паттернам debounce и throttle, кастомным событиям и базовому пониманию reflow. Предполагается, что читатель уже знает синтаксис JavaScript (тема `javascript-basics`). Упражнения решаются чистыми функциями: дерево DOM и поток событий смоделированы обычными объектами, поэтому код запускается в песочнице без реального браузера, а знание API проверяется через разбор кода и вопросы.

## Концепты

1. Дерево DOM и выборка узлов
2. Создание, изменение и удаление узлов
3. Атрибуты и свойства
4. Модель событий: погружение, цель, всплытие
5. addEventListener и объект Event
6. Делегирование событий
7. Паттерны событий и перерасчёт
