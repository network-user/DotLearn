# JavaScript с нуля

Первая JavaScript-тема каталога: значения и типы, let/const, условия и циклы, функции и стрелочные функции, строки и шаблонные литералы, массивы и объекты, map/filter/reduce, область видимости и замыкания. Семь концептов для новичка и начинающего фронтендера: знаний JavaScript не предполагается, достаточно общего представления о программировании. Все упражнения решаются чистыми функциями без DOM и сети.
