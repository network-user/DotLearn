# CSS-вёрстка: Flexbox и Grid

Продолжение вводной темы про HTML и CSS: здесь вёрстка разбирается вглубь. От честной арифметики box model и box-sizing через нормальный поток документа, display и позиционирование к двум системам раскладки - Flexbox и Grid - с реальными расчётами распределения места. Отдельные уроки про адаптивность (media- и container-queries, функции clamp/min/max) и про типичные ловушки, на которых спотыкаются даже опытные верстальщики: схлопывание margin, минимальный размер flex-элемента, новый контекст наложения. Тема теоретическая (runtime `none`): код показан в статических блоках, а упражнения просят предсказать поведение браузера и посчитать итоговые размеры, а не писать стили вслепую. После прохождения вы читаете чужую раскладку, объясняете, почему элемент занял именно столько места, и осознанно выбираете между Flexbox и Grid.

## Концепты

1. **Box model и box-sizing** · четыре слоя коробки, content-box против border-box, width/min-width/max-width, overflow.
2. **display и нормальный поток** · block, inline, inline-block; блочный и строчный контексты форматирования; за что отвечает поток.
3. **Позиционирование** · static, relative, absolute, fixed, sticky; containing block; z-index и контекст наложения.
4. **Flexbox** · главная и поперечная оси, justify/align, wrap, распределение места через flex-grow/shrink/basis с расчётами.
5. **Grid** · grid-template-columns/rows, единица fr, gap, grid-template-areas, repeat, minmax, auto-fit против auto-fill.
6. **Адаптивность** · mobile-first и media-queries, container queries, clamp/min/max, единицы вьюпорта.
7. **Типичные ловушки** · схлопывание margin, min-width flex-элемента, sticky без порога, z-index в новом контексте наложения.

## Предварительные требования

Тема `html-css-basics`: нужно понимать селекторы, каскад и базовый box model.
