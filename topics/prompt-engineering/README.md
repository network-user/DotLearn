# Промпт-инжиниринг

Практический курс по управлению большими языковыми моделями: из чего состоит промпт, как примеры и цепочки рассуждений меняют ответ, как получать надёжный структурный вывод, заземлять модель контекстом и RAG, защищаться от галлюцинаций и prompt injection и собирать продвинутые паттерны вроде ReAct и self-consistency.

Сгенерировано через `lesson-forge`.
