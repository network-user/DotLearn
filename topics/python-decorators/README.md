# Декораторы в Python

Декораторы шаг за шагом: функции как объекты, замыкания, синтаксис `@`, сохранение метаданных через `functools.wraps`, фабрики декораторов с аргументами, стек декораторов, классы и практические рецепты (memoize, retry, тайминг, регистрация).

Сгенерировано через `lesson-forge`.

## English (`en`)

Step-by-step decorators: functions as objects, closures, `@` syntax, preserving metadata with `functools.wraps`, decorator factories with arguments, decorator stacks, classes, and practical recipes (memoize, retry, timing, registration).
