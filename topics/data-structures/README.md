# Структуры данных

От массива и динамического роста list до стека, очереди, связного списка, хеш-таблицы, деревьев поиска, кучи и графов: как каждая структура устроена внутри, какие операции у неё дешёвые, а какие дорогие, и как реализовать её на чистом Python. Тема готовит к вопросам собеседований и опирается на интерактивные визуализации и упражнения на pyodide.

## English

# Data Structures in Python

From arrays and the dynamic growth of `list` to stacks, queues, linked lists, hash tables, binary search trees, heaps, and graphs: how each structure works internally, which operations are cheap and which are expensive, and how to implement it in plain Python. The topic prepares you for interview questions and relies on interactive visualizations and pyodide exercises.

2. Stacks and queues
3. Linked lists
4. Hash tables
5. Trees and binary search trees
6. Heaps and heapq
7. Graphs and their representations
