# React: компоненты и состояние

Практический курс по фундаменту React для тех, кто уже знает JavaScript. Вы узнаете, зачем нужен декларативный подход к интерфейсу, как читать и писать JSX, из чего складываются функциональные компоненты, как данные текут через пропсы сверху вниз и как состояние через `useState` делает интерфейс живым. Отдельно разбираем, что именно происходит при рендере, как показывать списки с ключами и ветвить разметку по условию, как обрабатывать события и как поднимать состояние в общего родителя, сохраняя однонаправленный поток данных. Хуки берём обзорно: глубокое погружение в `useEffect` и кастомные хуки живёт в соседней теме `react-hooks`.

Курс концептуальный и без выполнения кода (`runtime: none`): примеры на JSX разбираются в тексте и на интерактивных схемах, а закрепляются упражнениями на модель React, разбор JSX и предсказание того, что окажется на экране.

## Концепты

1. **Декларативный UI** - чем «описать результат» лучше, чем «командовать DOM по шагам»; формула UI как функции состояния.
2. **JSX** - разметка внутри JavaScript, фигурные скобки для выражений, атрибуты и `className`, один корень.
3. **Функциональные компоненты** - компонент как функция, возвращающая JSX; заглавные имена и композиция.
4. **Пропсы** - параметры компонента, поток данных сверху вниз, только для чтения, `children`.
5. **Состояние и useState** - память компонента между рендерами, пара «значение и сеттер», замена вместо мутации.
6. **Рендер и реактивность** - что происходит при изменении состояния: повторный вызов, сравнение деревьев, точечное обновление DOM.
7. **Списки и ключи** - вывод массива через `map`, зачем нужны ключи и почему индекс - плохой ключ.
8. **Условный рендеринг** - тернарный оператор, `&&`, ранний `return` и ловушка нуля.
9. **Обработка событий** - `onClick` и камелкейс, передача функции вместо вызова, аргументы и `preventDefault`.
10. **Подъём состояния** - общий родитель как владелец данных, значение вниз, коллбэк вверх, однонаправленный поток.

## Предварительные требования

Тема `javascript-basics`: переменные, функции и стрелочные функции, массивы и объекты, методы `map` и `filter`.
