# Хранилище Python: объекты под капотом

Погружение во внутреннее устройство памяти CPython: как переменные ссылаются на объекты, как работают подсчёт ссылок и сборщик мусора, как устроены хеш-таблицы dict и set, и какие оптимизации раскладки в памяти (`__slots__`, over-allocation, интернирование) применяет интерпретатор. Детали относятся к CPython 3.11+ как к конкретной реализации языка.
