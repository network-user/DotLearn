# TypeScript: типы для JavaScript

Первая TypeScript-тема каталога: зачем поверх JavaScript вообще нужны статические типы, базовые типы и вывод типов, объекты через `interface` и `type`, union-типы и narrowing, сигнатуры функций, generics, утилити-типы (`Partial`, `Pick`, `Omit`, `Record`), массивы и кортежи, тройка `unknown`/`any`/`never` и базовая настройка строгого `tsconfig`. Десять концептов для того, кто уже пишет на JavaScript и хочет ловить ошибки до запуска. Тема теоретическая: TypeScript в песочнице не исполняется, поэтому упражнения проверяют понимание типов и рантайм-результат кода после стирания аннотаций.

## Концепты

1. Зачем статические типы
2. Базовые типы и вывод типов
3. Объекты, интерфейсы и type
4. Union-типы и narrowing
5. Функции и сигнатуры
6. Массивы и кортежи
7. Generics: типы с параметром
8. Утилити-типы: Partial, Pick, Omit, Record
9. unknown, any и never
10. Строгий tsconfig
