# Python с нуля

Точка входа всего каталога: курс для человека, который никогда не программировал. Семь коротких уроков проводят от первой переменной до собственной функции: типы данных, строки, условия, списки и словари, циклы, срезы и функции. Каждый урок объясняет идею через бытовые аналогии, показывает живые примеры, которые можно менять и запускать прямо на странице, и закрепляет материал исполняемыми упражнениями на pyodide. Используется только стандартная библиотека Python, без сети и файловой системы. Предварительных знаний не требуется.
