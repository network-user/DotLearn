## English

# Python from Scratch

The entry point of the whole catalog: a course for someone who has never programmed before. Seven short lessons take you from your first variable to writing your own function: data types, strings, conditionals, lists and dictionaries, loops, slicing, and functions. Each lesson explains the idea through everyday analogies, shows live examples you can edit and run right on the page, and reinforces the material with executable exercises on pyodide. Only the Python standard library is used, with no network or filesystem access. No prior knowledge is required.

## Concepts

1. Variables and data types
2. Strings
3. Conditionals and branching
4. Lists and dictionaries
5. Loops
6. Slices and sequences
7. Functions
