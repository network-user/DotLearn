## Приоритет контекста

4. Код в затронутых файлах

## Триггеры

- «Обнови README» / «сгенерируй README» / правки документации → скилл `generate-readme` (стандарт DotCore).
- Глобальные изменения функционала (новые/удалённые команды, модули, пакеты, зависимости, смена архитектуры или runtime) → обнови `README.md` + `AGENTS.md` тем же скиллом, включая пересчёт LoC.
- «Добавь тему» / создать уроки, упражнения, курс → скилл `lesson-forge`. Не генерируй контент темы вручную.

Не дублируй содержимое AGENTS.md здесь - обновляй AGENTS.md через скилл.
