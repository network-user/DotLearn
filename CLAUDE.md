# Claude Code

Прочитай [AGENTS.md](AGENTS.md) перед любой задачей в этом репозитории - это единый источник правды по сборке, тестам, архитектуре и конвенциям.

## Приоритет контекста

1. Явный запрос пользователя
2. [AGENTS.md](AGENTS.md)
3. [README.md](README.md)
4. Код в затронутых файлах

## Триггеры

- «Обнови README» / «сгенерируй README» / правки документации → скилл `generate-readme` (стандарт DotCore).
- «Обнови AGENTS.md» / «синхронизируй правила проекта» (только правила, без README) → скилл `sync-project-rules`.
- Глобальные изменения функционала (новые/удалённые команды, модули, пакеты, зависимости, смена архитектуры или runtime) → обнови `README.md` через `generate-readme` (с пересчётом LoC) и `AGENTS.md` через `sync-project-rules`.
- «Добавь тему» / создать уроки, упражнения, курс → скилл `lesson-forge`. Не генерируй контент темы вручную.

Не дублируй содержимое AGENTS.md здесь - обновляй AGENTS.md через скилл.
