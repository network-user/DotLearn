# alt и title у img: доступность, SEO и всплывающие подсказки

## Суть

`alt` задаёт текстовую замену изображения для AT и случая, когда картинка не загрузилась. `title` добавляет опциональную подсказку при наведении, но не заменяет `alt` и редко читается скринридерами как основное имя.

## Что ответить на собесе

- `alt` обязателен для смысловых `<img>`; для декора без информации ставлю пустой `alt=""`.
- Описываю содержание, а не файл: «логотип DotLearn», не «logo.svg».
- `title` использую для дополнительного контекста, не дублируя `alt` слово в слово.
- Упоминаю, что поисковики индексируют `alt`, а `title` на img почти не влияет на SEO.
- Честно добавляю: длинный `title` на тач-устройствах бесполезен, потому что hover нет.

## Как это работает

Скринридер берёт **accessible name** из `alt`. Если атрибут пустой и роль decorative, изображение пропускают в озвучке.

```html
<img
  src="/covers/react-hooks.webp"
  alt="Схема жизненного цикла хука useEffect в React"
  width="640"
  height="360"
  loading="lazy"
/>

<img src="/patterns/grid.svg" alt="" role="presentation" />
```

Второй пример: чисто декоративный фон, AT его игнорирует.

`title` рендерится как native tooltip на desktop при `:hover`:

```html
<img
  src="/team/photo.jpg"
  alt="Команда на митапе, четыре человека у стенда"
  title="Фото с Frontend Meetup 2025"
/>
```

Для информативных картинок текст из `alt` должен передавать смысл без просмотра. Для диаграмм иногда дублируют описание в видимом `<figcaption>` рядом в `<figure>`.

## Подводные камни

**Пустой alt на контенте.** Пользователь слепой не узнает, что на слайде график продаж.

**Keyword stuffing в alt.** Спам ключевыми словами вредит UX и может навредить SEO.

**title вместо alt.** Подсказка не показывается при ошибке загрузки и не является основным именем для AT.

**Одинаковый alt на всех иконках.** «Иконка» на каждой кнопке не даёт контекста; лучше скрытый текст или `aria-label` на кнопке.

**CSS background-image без текстовой альтернативы.** Фон не имеет `alt`; смысл нужно дать в разметке рядом.
