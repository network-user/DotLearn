# Функция как аргумент: что значит «первоклассность» функций в Python

## Суть

Да, в Python функцию можно передать в другую функцию как обычное значение. Функции здесь - объекты первого класса, поэтому с ними допустимо всё, что и с любыми данными.

## Что ответить на собесе

Функции в Python - объекты первого класса. Их можно присваивать переменным, складывать в списки и словари, возвращать из других функций и передавать аргументами. Именно на этом держатся `map`, `filter`, `sorted(key=...)`, декораторы и колбэки. Передавая поведение параметром, вы выносите изменяемую часть наружу и переиспользуете общий каркас вместо копирования кода.

## Как это работает

Имя функции без скобок - это ссылка на объект-функцию, а не её вызов. Эту ссылку и передают:

```python
def square(x):
    return x * x

numbers = [1, 2, 3, 4, 5]

squared = map(square, numbers)
print(list(squared))   # [1, 4, 9, 16, 25]
```

`map(square, numbers)` получает саму функцию `square` (без `()`) и применяет её к каждому элементу. Тот же приём работает в сортировке и фильтрации, где функция задаёт критерий:

```python
words = ['ccc', 'a', 'bb']

print(sorted(words, key=len))                  # ['a', 'bb', 'ccc']
print(list(filter(lambda w: len(w) > 1, words)))  # ['ccc', 'bb']
```

Можно написать и собственную функцию высшего порядка, которая принимает действие параметром:

```python
def apply_twice(func, value):
    return func(func(value))

print(apply_twice(lambda x: x + 3, 10))   # 16
```

## Подводные камни

- `square` - ссылка на функцию, `square()` - её немедленный вызов. Передадите `square()` вместо `square` - в аргумент попадёт результат вызова (или ошибка про отсутствующие параметры).
- `map` и `filter` возвращают ленивый итератор, а не список. Пройти по нему можно один раз; для печати или повторного обхода оборачивайте в `list()`.
- Передавая метод объекта (`obj.method`), вы передаёте связанный метод вместе с `self` - это уже не «голая» функция.
- Лямбды удобны, но злоупотребление ими ухудшает читаемость и затрудняет отладку - в трейсбеке у них нет осмысленного имени.
