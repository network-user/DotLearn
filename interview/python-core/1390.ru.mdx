# Как поменять местами две переменные одной строкой?

## Суть

Идиома `a, b = b, a` меняет значения местами без временной переменной за счёт упаковки правой части в кортеж и его распаковки в левую.

## Что ответить на собесе

В Python обмен пишется в одну строку: `a, b = b, a`. Промежуточная переменная не нужна, потому что справа сначала целиком вычисляется кортеж `(b, a)` из текущих значений, и только потом его элементы распаковываются в `a` и `b`. Это идиоматично, читаемо и работает для любого числа переменных.

## Как это работает

Присваивание в Python вычисляет правую часть до того, как что-либо записать в левую. Поэтому порядок такой:

1. Справа собирается кортеж `(b, a)` из текущих значений.
2. Слева стоят целевые имена `a` и `b`.
3. Готовый кортеж распаковывается: первый элемент уходит в `a`, второй - в `b`.

```python
x = 5
y = 10

x, y = y, x

print("x:", x)  # x: 10
print("y:", y)  # y: 5
```

Тот же приём масштабируется на любое количество переменных, например `a, b, c = c, a, b`.

## Подводные камни

- Важна именно атомарность вычисления правой части: значения берутся до присваивания, поэтому порядок имён слева и справа не «перетирает» данные.
- При распаковке число имён слева должно совпадать с числом значений справа, иначе `ValueError`.
- В цепочках с индексами вроде `a[i], a[j] = a[j], a[i]` индексы тоже вычисляются по правилам кортежа - это безопасно для свапа элементов списка.
- Не путайте с `a = b = value`: это присваивает одно значение обоим, а не меняет местами.
