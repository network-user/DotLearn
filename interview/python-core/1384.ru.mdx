# Как собрать список одной строкой через списковое включение?

## Суть

Списковое включение (list comprehension) - это идиома Python, которая в одном выражении строит новый список из любого итерируемого источника. По сути это свёрнутый цикл `for`, в который можно встроить и преобразование элемента, и отбор по условию.

## Что ответить на собесе

List comprehension декларативно описывает построение списка: `[выражение for элемент in источник if условие]`.

- Читается слева направо: «возьми каждый элемент источника, при необходимости отфильтруй и применённое выражение положи в результат».
- Короче и выразительнее ручного цикла с `append`.
- Обычно немного быстрее: цикл крутится на уровне интерпретатора, а не в Python-коде с вызовом метода на каждой итерации.

## Как это работает

Полная форма состоит из четырёх частей:

```python
new_list = [expression for item in iterable if condition]
```

- `expression` - что положить в результат для текущего элемента;
- `item` - переменная, по очереди принимающая каждое значение из источника;
- `iterable` - то, по чему идём перебором;
- `condition` - необязательный фильтр; без него берутся все элементы.

Простой случай без условия - квадраты чисел:

```python
squares = [x ** 2 for x in range(10)]
# [0, 1, 4, 9, 16, 25, 36, 49, 64, 81]
```

С фильтром оставим только чётные:

```python
evens = [x for x in range(10) if x % 2 == 0]
# [0, 2, 4, 6, 8]
```

Вложенные циклы перечисляются слева направо в том же порядке, что и обычные `for`:

```python
pairs = [(x, y) for x in range(3) for y in range(3)]
# [(0, 0), (0, 1), (0, 2), (1, 0), ... (2, 2)]
```

## Подводные камни

- Перегруженное включение с тремя циклами и парой условий читается хуже обычного цикла - в таком случае разворачивайте его обратно.
- Тернарный оператор и фильтр - разные вещи: `[x if x else 0 for x in data]` ставит условие в выражение, а `[x for x in data if x]` фильтрует. Местоположение `if` меняет смысл.
- Включение материализует весь список в памяти. Если он большой и нужен лишь один проход, берите генераторное выражение `(... for ...)` в круглых скобках.
- Переменная цикла внутри включения не утекает в окружающую область видимости (в отличие от Python 2) - не рассчитывайте увидеть `item` после включения.
