# Как получать позицию элемента прямо в цикле по последовательности?

## Суть

Оборачивайте последовательность в `enumerate()` - на каждом шаге она отдаёт пару «индекс, элемент», и ручной счётчик становится не нужен.

## Что ответить на собесе

Идиоматичный способ - `for i, value in enumerate(seq)`. Это лучше, чем `for i in range(len(seq))` с обращением `seq[i]`, потому что код читается чище и не зависит от того, поддерживает ли объект индексацию. У `enumerate` есть второй аргумент `start`, если нумерация должна начинаться не с нуля.

## Как это работает

`enumerate()` принимает любой итерируемый объект и возвращает ленивый итератор. На каждом шаге он выдаёт кортеж `(индекс, элемент)`, который распаковывается в две переменные цикла.

```python
fruits = ['apple', 'banana', 'cherry']

for index, value in enumerate(fruits):
    print(index, value)
```

Вывод:

```text
0 apple
1 banana
2 cherry
```

Если нужно начать счёт с единицы, передайте `start`:

```python
for number, value in enumerate(fruits, start=1):
    print(number, value)
```

## Подводные камни

- `enumerate` возвращает итератор, а не список: повторно пройтись по нему не получится, для этого оберните в `list()`.
- Первый элемент кортежа - индекс, второй - значение. Перепутать порядок распаковки - частая ошибка.
- Аргумент `start` смещает только нумерацию, но не сами элементы; индексация исходной последовательности при этом не меняется.
