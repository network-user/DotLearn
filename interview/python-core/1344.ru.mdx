# Что превращает обычный объект в итератор?

## Суть

Объект становится итератором, когда у него есть пара методов: `__iter__()` (возвращает сам объект) и `__next__()` (отдаёт очередной элемент и поднимает `StopIteration`, когда они закончились).

## Что ответить на собесе

Итератор — это объект, реализующий протокол итерации из двух методов. `__iter__()` обязан вернуть сам итератор, а `__next__()` на каждый вызов отдаёт следующий элемент. Когда выдавать больше нечего, `__next__()` бросает `StopIteration`, и цикл `for` корректно завершается. Важно не путать итерируемое (есть только `__iter__`) и итератор (есть оба метода): список итерируем, но сам по себе не итератор — его `__iter__` создаёт отдельный итератор.

## Как это работает

Цикл `for` под капотом сначала вызывает `iter(obj)`, то есть `__iter__()`, чтобы получить итератор. Затем он в цикле дёргает `next()` — то есть `__next__()` — пока не поймает `StopIteration`.

```python
class CountUpTo:
    def __init__(self, limit):
        self.limit = limit
        self.current = 0

    def __iter__(self):
        return self

    def __next__(self):
        if self.current < self.limit:
            value = self.current
            self.current += 1
            return value
        raise StopIteration


for n in CountUpTo(3):
    print(n)
```

Этот код напечатает `0`, `1`, `2`. Поскольку `__iter__` возвращает `self`, состояние перебора хранится прямо в экземпляре. Так можно описывать и потенциально бесконечные последовательности, не держа их в памяти целиком.

## Подводные камни

- `__iter__` обязан вернуть итератор. Если он вернёт что-то без `__next__`, цикл `for` упадёт с `TypeError`.
- Одноразовость: если `__iter__` возвращает `self`, итератор исчерпывается после первого прохода. Второй цикл `for` по тому же экземпляру не выдаст ничего.
- Забытый `raise StopIteration` приведёт к бесконечному циклу.
- Итерируемое (`__iter__`) и итератор (`__iter__` + `__next__`) — разные вещи. Чтобы объект можно было обходить много раз, `__iter__` должен возвращать каждый раз новый итератор.
