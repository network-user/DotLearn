# Class inheritance in Java for AQA

## In short

Inheritance links a subclass to a superclass with `extends`: the child gets non-private fields and methods, can extend the API, and override behavior. In test frameworks and page object layers this is how you reuse base steps and run polymorphic fixtures.

## How to answer in an interview

- Base class with shared helpers; subclasses for specific services or pages.
- `@Override` on overridden methods; `super()` in the constructor to initialize the parent.
- Polymorphism: a parent-typed variable references a child object.
- Composition beats deep hierarchies in test code.
- Honestly: Java has no multiple class inheritance, only one `extends`.

## How it works

```
class Vehicle {
  protected String label;

  Vehicle(String label) {
    this.label = label;
  }

  String describe() {
    return "vehicle " + label;
  }
}

class Truck extends Vehicle {
  Truck(String label) {
    super(label);
  }

  @Override
  String describe() {
    return "truck " + label;
  }
}
```

In a test:

```
Vehicle unit = new Truck("cargo");
assertEquals("truck cargo", unit.describe());
```

Python mirror for a pytest base:

```python
class ApiClient:
    def __init__(self, base_url: str) -> None:
        self.base_url = base_url

    def ping(self) -> str:
        return f"ok {self.base_url}"


class OrdersClient(ApiClient):
    def ping(self) -> str:
        return f"orders {self.base_url}"


def test_inheritance_polymorphism():
    client: ApiClient = OrdersClient("https://api.test")
    assert client.ping() == "orders https://api.test"
```

## Pitfalls

**private is not inherited.** The child cannot access the parent's private fields directly.

**Parent constructor.** Without an explicit `super(...)`, the compiler requires a no-arg superclass constructor.

**Deep trees.** A page object with five inheritance levels breaks when the UI changes.

**final methods.** Cannot be overridden; a test stub via subclass will not work.

**Liskov violations.** A child that tightens a method contract breaks polymorphic tests.
