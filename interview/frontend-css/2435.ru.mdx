# Что такое CSS-in-JS и когда его выбирают

## Суть

CSS-in-JS пишет стили внутри JavaScript/TSX: styled-компоненты, объектные стили или runtime injection. Стили привязаны к props и state, классы часто генерируются автоматически.

## Что ответить на собесе

- Плюсы: colocation с компонентом, динамика от props, scoped class names без ручных modules.
- Минусы: runtime cost, SSR setup, DevTools иногда менее прозрачны, риск дублирования CSS в бандле.
- Библиотеки: Emotion, styled-components, Stitches; zero-runtime варианты: Linaria, Vanilla Extract.
- Для design system tokens комбинирую theme object + CSS variables.
- Честно: в 2020-х многие команды вернулись к modules + tokens; CSS-in-JS не default для каждого проекта.

## Как это работает

```ts
import styled from "@emotion/styled";

interface PillProps {
  tone?: "neutral" | "danger";
}

const Pill = styled.span<PillProps>`
  display: inline-flex;
  align-items: center;
  padding: 0.25rem 0.625rem;
  border-radius: 999px;
  font-size: 0.8125rem;
  font-weight: 600;
  background: ${({ tone }) =>
    tone === "danger" ? "#fee2e2" : "#e2e8f0"};
  color: ${({ tone }) =>
    tone === "danger" ? "#991b1b" : "#334155"};
`;

export function StatusBadge({ failed }: { failed: boolean }) {
  return <Pill tone={failed ? "danger" : "neutral"}>Статус</Pill>;
}
```

Объектный стиль (inline object libs):

```ts
const rowStyle = {
  display: "grid",
  gridTemplateColumns: "1fr auto",
  gap: "0.5rem",
  alignItems: "center",
} as const;
```

Zero-runtime compile time:

```ts
import { css } from "@linaria/core";

export const shell = css`
  max-width: 48rem;
  margin-inline: auto;
  padding: 1rem;
`;
```

## Подводные камни

**Specificity wars с global CSS.** Inject порядок stylesheet влияет на override.

**SSR и hydration mismatch.** Разный class hash на сервере и клиенте даёт warning.

**Производительность на списках.** Styled component на каждый item умножает injection.

**Отсутствие static extract.** Runtime libs раздувают JS на mobile.

**Миграция.** Смешение modules + CSS-in-JS в одном UI-kit усложняет theming.
