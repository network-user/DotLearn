# Flexbox: свойства контейнера и элементов

## Суть

Flexbox раскладывает прямых потомков вдоль главной и поперечной оси. Контейнер задаёт направление, перенос и выравнивание; элементы управляют ростом, сжатием и собственным align.

## Что ответить на собесе

- Контейнер: `display:flex`, `flex-direction`, `flex-wrap`/`flex-flow`, `justify-content`, `align-items`, `align-content`, `gap`.
- Элементы: `flex-grow`, `flex-shrink`, `flex-basis` (шорткат `flex`), `align-self`, `order`.
- `justify-*` по главной оси, `align-*` по поперечной; при `column` оси меняются местами.
- `flex: 1` часто значит `1 1 0%` и равномерно делит свободное место.
- Честно добавляю: `order` меняет визуальный порядок, не таб-фокус; для a11y опасно без продуманного DOM.

## Как это работает

Toolbar с кнопками:

```css
.toolbar {
  display: flex;
  flex-wrap: wrap;
  align-items: center;
  justify-content: space-between;
  gap: 0.75rem;
}

.toolbar__actions {
  display: flex;
  gap: 0.5rem;
  margin-left: auto;
}

.toolbar__btn {
  flex: 0 0 auto;
}
```

`margin-left: auto` на группе действий прижимает её к концу главной оси без `justify-content: flex-end` на всём баре.

Карточка с растягивающимся телом:

```css
.card {
  display: flex;
  flex-direction: column;
  min-height: 12rem;
}

.card__body {
  flex: 1 1 auto;
}

.card__footer {
  flex-shrink: 0;
}
```

`flex-basis` задаёт стартовый размер до распределения free space. `flex-shrink: 0` защищает футер от сжатия при нехватке высоты.

## Подводные камни

**min-width: auto на flex-item.** Длинный текст не сжимается; фикс через `min-width: 0` на item.

**align-items: stretch и вложенные img.** Картинка растягивается искажённо без `align-self: flex-start`.

**flex-basis: 100% + wrap.** Элемент занимает всю строку и ломает сетку из нескольких колонок.

**gap в старых Safari.** Нужен fallback margin для критичных лейаутов или проверка baseline браузеров.

**column + justify-content: center на коротком экране.** Контент обрезается сверху при overflow; лучше `align-items: stretch` и внутренний scroll.
