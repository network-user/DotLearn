# Что меняют свойства flexbox у контейнера и элементов

## Суть

Flexbox делит контейнер на главную и поперечную оси. Свойства контейнера задают направление, перенос и выравнивание группы; свойства элементов распределяют свободное место и базовый размер.

## Что ответить на собесе

- На контейнере: `display:flex`, `flex-direction`, `flex-wrap`, `justify-content` (главная ось), `align-items`/`align-content` (поперечная).
- На элементах: `flex-grow`, `flex-shrink`, `flex-basis`, shorthand `flex`, плюс `align-self` и `order`.
- `justify-content` работает по главной оси; при `flex-direction: column` «горизонталь» становится поперечной.
- Для равных колонок без процентов использую `flex: 1 1 0` или `flex-basis: 0`.
- Честно: flexbox отлично для одномерных рядов; двумерные сетки чаще отдаю Grid.

## Как это работает

Toolbar с переносом и равномерным распределением:

```ts
const toolbarStyle = {
  display: "flex",
  flexDirection: "row" as const,
  flexWrap: "wrap" as const,
  justifyContent: "space-between",
  alignItems: "center",
  gap: "0.75rem",
};

const chipStyle = {
  flex: "1 1 120px",
  minWidth: "6rem",
};
```

```css
.toolbar {
  display: flex;
  flex-direction: row;
  flex-wrap: wrap;
  justify-content: space-between;
  align-items: center;
  gap: 0.75rem;
}

.chip {
  flex: 1 1 120px;
  min-width: 6rem;
  padding: 0.5rem 0.75rem;
  border-radius: 999px;
  background: color-mix(in srgb, currentColor 8%, transparent);
}
```

</figure>

`flex-grow: 1` забирает остаток строки; `flex-shrink: 0` защищает минимальную ширину иконки.

## Подводные камни

**margin:auto на flex-item.** Съедает свободное место неожиданным образом, особенно в колонке.

**align-items: stretch и фиксированная высота детей.** Дети тянутся, но контент внутри может не центрироваться.

**flex-basis: auto и min-width.** Элемент не сжимается ниже контента без явного `min-width: 0`.

**gap vs margin.** Старые Safari без gap ломают «карточные» сетки, если не проверить support.

**100% height в column flex.** Родитель без заданной высоты не даст ребёнку «растянуться» по вертикали.
