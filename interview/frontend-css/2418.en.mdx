# Responsive layouts with Flexbox and CSS Grid

## In short

Flexbox lays out items along one axis with wrapping; Grid defines two-dimensional tracks. Card galleries often need only `auto-fill`/`minmax`; full page layouts usually favor Grid.

## How to answer in an interview

- Flex: `display:flex`, `flex-wrap:wrap`, `flex: 1 1 280px` or `flex-basis` plus `min-width` for even cards.
- Grid: `grid-template-columns: repeat(auto-fit, minmax(16rem, 1fr))`, `gap` instead of margin hacks.
- Media queries still help, but `minmax`/`auto-fit` remove some breakpoints.
- `subgrid` (where supported) aligns nested cards to the parent grid.
- I honestly add that mixing Flex and Grid for the same gallery adds complexity; pick one tool per job.

## How it works

**Flex catalog:**

```css
.card-grid {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.card-grid > * {
  flex: 1 1 18rem;
  max-width: 100%;
}
```

Items grow from ~288px and wrap when space runs out. `gap` sets rhythm without negative margins.

**Grid with auto-fit:**

```css
.topic-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(14rem, 1fr));
  gap: 1.25rem;
}
```

`auto-fit` collapses empty tracks; `minmax` prevents columns narrower than the minimum.

Page shell:

```css
.app-shell {
  display: grid;
  grid-template-columns: minmax(12rem, 240px) 1fr;
  min-height: 100dvh;
}

@media (max-width: 48rem) {
  .app-shell {
    grid-template-columns: 1fr;
  }
}
```

Sidebar and main on Grid; inside main, cards use `topic-grid` again.

## Pitfalls

**Percentage flex-basis without gap.** Two 50% items plus margin break the row; use `gap` or `calc`.

**auto-fill vs auto-fit.** `auto-fill` keeps empty columns; rows look hollow.

**Grid without minmax on small screens.** Fixed `200px 200px 200px` causes horizontal scroll.

**Equal card height.** Flex/Grid do not stretch inner content; use an inner flex column with `margin-top: auto` on the card footer.

**DOM order vs a11y.** Flex `order` changes visual order, not focus order; keyboard flows suffer if abused.
