# Какие CSS-свойства вызывают reflow и repaint

## Суть

Reflow затрагивает геометрию: размеры, отступы, позиционирование, шрифты, display. Repaint перерисовывает пиксели без смены layout (color, background, visibility частично). Композиторные transform/opacity обычно не трогают layout.

## Что ответить на собесе

- Layout-триггеры: width/height, margin/padding/border, top/left, font-size, line-height, display, position, float, overflow, content.
- DOM-операции: insert/remove, изменение text node, чтение layout после write.
- Только paint: color, outline, background, box-shadow (без смены размеров).
- Композит: transform, opacity, filter (осторожно с дочерними fixed).
- Честно добавляю: таблица «что вызывает reflow» упрощена; браузер батчит, но forced sync layout всё равно дорог.

## Как это работает

Pipeline: style → layout → paint → composite.

```ts
const box = document.querySelector<HTMLElement>(".panel");
if (!box) return;

box.style.width = "320px";
box.style.padding = "1rem";
box.style.transform = "scale(1.02)";
box.style.color = "#0f172a";
```

Первые два свойства могут вызвать reflow соседей; `transform` и `color` ограничатся композитом/paint без пересчёта ширины сетки.

Чтение после записи форсирует layout:

```ts
box.style.width = "400px";
const w = box.offsetWidth;
box.style.height = `${w / 2}px`;
```

Между строками браузер вынужден синхронно пересчитать layout.

Типичные layout-свойства в списке для собеса:

```
width, height, min-/max-*, margin, padding, border-width,
top, left, right, bottom, font-size, line-height,
display, position, float, clear, overflow, vertical-align
```

Добавление/удаление DOM-узла и смена `document.fonts` тоже инвалидируют layout.

## Подводные камни

**offsetWidth в scroll handler.** Каждый scroll event + read = jank.

**classList.toggle на body.** Смена `overflow: hidden` для modal блокирует scroll и reflow всей страницы.

**box-shadow blur огромный.** Paint дорогой даже без reflow.

**filter: blur() на большом блоке.** GPU fill-rate, не layout, но FPS падает.

**Ложная безопасность translate.** `transform` не reflow, но соседний `height: auto` анимировать нельзя так же легко.
