# Какие значения бывают у CSS position

## Суть

Пять ключевых значений: `static` (default), `relative`, `absolute`, `fixed`, `sticky`. Они задают, участвует ли элемент в потоке, относительно чего считаются offsets и создаётся ли containing block для потомков.

## Что ответить на собесе

- static: normal flow, offsets игнорируются.
- relative: смещение + positioning context, slot в потоке сохраняется.
- absolute/fixed: вне потока; absolute от non-static предка, fixed от viewport (с оговорками transform).
- sticky: relative до threshold scroll, затем «липнет» в scroll container.
- Честно: `position: sticky` ломается при `overflow:hidden` на предке.

## Как это работает

```css
.toolbar { position: sticky; top: 0; z-index: 10; }
.drawer { position: fixed; inset: 0; }
.popover {
  position: absolute;
  top: calc(100% + 0.25rem);
  right: 0;
}
.anchor { position: relative; }
.badge { position: absolute; top: -0.25rem; right: -0.25rem; }
```

```ts
type PositionMode =
  | "static"
  | "relative"
  | "absolute"
  | "fixed"
  | "sticky";

function setPosition(el: HTMLElement, mode: PositionMode) {
  el.style.position = mode;
}
```

Краткая таблица поведения:

```
static   · flow, no offsets
relative · flow + offset + context
absolute · out of flow, ancestor anchor
fixed    · out of flow, viewport anchor
sticky   · hybrid within scrollport
```

## Подводные камни

**sticky без top/left.** Без threshold sticky не активируется.

**fixed в transformed modal.** Containing block не viewport.

**absolute height collapse.** Родитель без height схлопывается.

**z-index только на positioned.** static игнорирует z-index.

**Путаница sticky vs fixed header.** sticky живёт в section, fixed глобален.
