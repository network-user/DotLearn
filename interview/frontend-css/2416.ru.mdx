# Box model: content, padding, border, margin и border-box

## Суть

Каждый элемент это прямоугольник из content, padding, border и margin. По умолчанию `width`/`height` задают только content-box; `box-sizing: border-box` включает padding и border в заявленные размеры.

## Что ответить на собесе

- Content: область текста и медиа; padding: отступ внутри границы; border: рамка; margin: внешний зазор до соседей.
- В content-box итоговая ширина = width + padding + border.
- `border-box` делает width полной внутренней шириной блока включая padding и border.
- Margin не входит в размер элемента, но участвует в схлопывании (margin collapse).
- Честно добавляю: глобальный `* { box-sizing: border-box }` упрощает вёрстку, но нужно помнить про replaced elements (`img`, `input`).

## Как это работает

Два режима для блока 200px:

```css
.box-content {
  box-sizing: content-box;
  width: 200px;
  padding: 16px;
  border: 4px solid #cbd5e1;
}

.box-border {
  box-sizing: border-box;
  width: 200px;
  padding: 16px;
  border: 4px solid #cbd5e1;
}
```

У `.box-content` видимая ширина ≈ 200 + 32 + 8 = 240px. У `.box-border` внешняя ширина остаётся 200px, content сжимается.

Практика в проектах:

```css
*,
*::before,
*::after {
  box-sizing: border-box;
}
```

Flex/Grid раскладывают **border-box** размеры по трекам; процентная ширина считается от родителя, padding не «раздувает» колонку за 100%.

DevTools вкладка Layout рисует слои margin (оранжевый), border, padding, content: удобно ловить лишние отступы.

## Подводные камни

**100% width + padding в content-box.** Элемент вылезает за родителя без `border-box`.

**margin collapse.** Вертикальные margin соседних блоков схлопываются; `padding`/`border` на родителе или `display: flow-root` разрывают эффект.

**min-height и проценты.** Процентная высота требует заданной высоты родителя; иначе поведение неочевидно.

**outline не в box model.** Outline не влияет на layout, но перекрывает соседей визуально.

**box-sizing на replaced elements.** `img` с фиксированной width и border-box ведёт себя предсказуемее в карточках с рамкой.
