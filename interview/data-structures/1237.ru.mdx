# Во сколько обходится сортировка пузырьком по времени и памяти?

## Суть

По времени пузырёк квадратичный: O(n^2) в среднем и худшем случаях, O(n) на уже упорядоченном массиве с ранним выходом. По памяти - O(1), потому что сортирует на месте.

## Что ответить на собесе

Bubble sort - стабильная сортировка на месте: время O(n^2), память O(1).

- Средний и худший случай O(n^2): два вложенных прохода дают порядка n*n сравнений.
- Лучший случай O(n) достижим с флагом, который прерывает цикл, когда за проход не было ни одной перестановки.
- Память O(1): кроме пары временных переменных, ничего не нужно.
- На практике её не применяют из-за квадратичной асимптотики, но она удобна как учебный пример.

## Как это работает

Алгоритм многократно проходит по массиву, сравнивает соседей и меняет их местами при нарушении порядка. За каждый проход самый большой из оставшихся элементов "всплывает" в конец, поэтому область сортировки сужается.

```python
def bubble_sort(arr):
    n = len(arr)
    for i in range(n):
        swapped = False
        for j in range(0, n - i - 1):
            if arr[j] > arr[j + 1]:
                arr[j], arr[j + 1] = arr[j + 1], arr[j]
                swapped = True
        if not swapped:
            break
    return arr
```

Флаг `swapped` - ключевой штрих: если за полный внутренний проход обмена не случилось, массив уже упорядочен и внешний цикл прерывается. Именно это даёт O(n) в лучшем случае.

## Подводные камни

- Без флага `swapped` лучшего случая O(n) не будет - алгоритм всегда отработает все n проходов и останется O(n^2).
- Граница внутреннего цикла - `n - i - 1`: на i-м проходе последние i элементов уже на местах, повторно их трогать не нужно.
- Пузырёк **стабилен** (равные элементы сохраняют исходный порядок), потому что меняет местами только при строгом `>`. Замена на `>=` стабильность сломает.
- O(1) по памяти - частая ловушка: люди путают её с временной сложностью. Это разные оси оценки.
- Для реальных данных берут merge sort или quicksort; пузырёк проигрывает им на порядки уже на тысячах элементов.
