# В чём разница между массивом и связным списком

## Суть

Обе структуры хранят последовательность, но по-разному раскладывают её в памяти. Массив - это непрерывный блок с мгновенным доступом по индексу за O(1). Связный список - цепочка узлов со ссылками, дешёвый на вставку/удаление, но медленный на доступ за O(n).

## Что ответить на собесе

Массив быстр на доступ по индексу, связный список - на вставку и удаление; выбор это компромисс.

- Массив лежит в памяти подряд, адрес элемента вычисляется арифметикой - доступ по индексу за O(1), но вставка/удаление в середину O(n).
- Связный список хранит узлы где угодно, связывая их ссылками: вставка и удаление за O(1), если позиция известна.
- Зато доступ к k-му элементу списка требует прохода с головы за O(n) и тратит память на сами ссылки.
- Итог: массив для частого доступа по индексу, список для частых вставок и удалений.

## Как это работает

Массив адресуется по индексу напрямую, без обхода:

```python
array = [1, 2, 3, 4, 5]
third = array[2]   # 3, доступ за O(1)
array[2] = 10      # [1, 2, 10, 4, 5]
```

Связный список обходят, переходя по ссылке `next` от узла к узлу:

```python
class Node:
    def __init__(self, data):
        self.data = data
        self.next = None


node1 = Node(1)
node2 = Node(2)
node3 = Node(3)
node1.next = node2
node2.next = node3

current = node1
while current is not None:
    print(current.data)   # 1, затем 2, затем 3
    current = current.next
```

## Подводные камни

- Сложность O(1) на вставку/удаление в списке справедлива, только если узел-сосед уже найден; поиск этого места сам по себе стоит O(n).
- Список тратит дополнительную память на ссылки и хуже для кэша процессора: узлы разбросаны, тогда как массив лежит подряд и читается быстрее на практике.
- Встроенный `list` в Python - это динамический массив, а не связный список: доступ по индексу O(1), но вставка в начало (`insert(0, x)`) - O(n).
- «Фиксированный размер» относится к низкоуровневым массивам (например, из модуля `array` или в C); питоновский `list` растёт сам, амортизированно добавляя ёмкость.
