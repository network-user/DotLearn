# You need a plain list of a dict's keys - how do you get it?

## In short

`dict.keys()` returns not a list but a lightweight view of the keys. If you specifically need a `list`, wrap the call in `list()`: `list(my_dict.keys())`.

## How to answer in an interview

The `.keys()` method returns a `dict_keys` view object, not a list. It is a dynamic "view" of the keys: it doesn't copy the data and reflects changes to the dictionary in real time. You can iterate over it directly in a loop, but if I need a real list - with indexing, slicing, or the ability to modify it independently of the dictionary - I explicitly convert it via `list(my_dict.keys())`. The same applies to `.values()` and `.items()`: all three return views, not lists.

## How it works

```python
my_dict = {'name': 'Alice', 'age': 30, 'city': 'New York'}

keys_view = my_dict.keys()      # dict_keys view object
keys_list = list(keys_view)     # plain list

print(keys_list)                # ['name', 'age', 'city']
```

A view is a window into the dictionary. It is live: add a key to the dictionary and the view shows it immediately.

```python
d = {'a': 1}
view = d.keys()
d['b'] = 2
print(list(view))   # ['a', 'b'] - the view saw the new key
```

`list()`, on the other hand, takes a snapshot: the resulting list is frozen and no longer depends on the dictionary. The order of keys in modern Python (3.7+) matches insertion order.

## Pitfalls

- `dict.keys()[0]` won't work - a view has no indexing. First `list(...)`, then the index.
- A view is **live**: if you iterate over it while modifying the dictionary at the same time, you'll hit `RuntimeError: dictionary changed size during iteration`. Take a snapshot via `list()` in advance.
- Converting to a list copies the keys into memory - on huge dictionaries that's wasted overhead if you just need to loop over them (in which case the view alone is enough).
- `dict_keys` behaves like a set and supports the `&`, `|`, `-` operations. A list has nothing of the sort - that's sometimes a reason to keep the view instead.
