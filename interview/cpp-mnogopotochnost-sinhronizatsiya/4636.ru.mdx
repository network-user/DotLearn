# Чем процесс отличается от потока?

## Суть

Процесс — изолированный экземпляр программы со своим адресным пространством, дескрипторами и учётом ОС. Поток — единица выполнения внутри процесса: потоки делят память и ресурсы процесса, но имеют собственный стек и контекст CPU.

## Что ответить на собесе

- Процесс падает отдельно; потоки одного процесса делят адресное пространство и могут уронить весь процесс.
- Создание процесса дороже (fork/exec, новый PID); поток дешевле, но требует синхронизации общих данных.
- Межпроцессное взаимодействие: pipe, socket, shared memory с ограничениями; между потоками достаточно глобальных переменных под mutex.
- В C++ потоки — `std::thread`, jthread; процессы — OS API (`fork`, `CreateProcess`), не STL.
- Честно: на Windows нет POSIX `fork`; сравнение строю на общих идеях ОС, а не на одном API.

## Как это работает

Процесс получает отдельную виртуальную память. Потоки одного процесса видят те же глобальные и кучу.

```cpp
#include <iostream>
#include <thread>

int sharedTotal = 0;

void worker(int id) {
    sharedTotal += id;
    std::cout << "thread " << id << " sum=" << sharedTotal << '\n';
}

int main() {
    std::thread a(worker, 1);
    std::thread b(worker, 2);
    a.join();
    b.join();
    return 0;
}
```

Два потока пишут в `sharedTotal` без защиты (на собесе упомяну гонку). Отдельный процесс не увидел бы эту переменную без IPC. Переключение потоков дешевле смены контекста процесса, зато ошибка в одном потоке (например, повреждение кучи) затрагивает всех.

## Подводные камни

- **Гонки в общей памяти.** Потоки делают data race тривиальным; процессы изолированы, но IPC сложнее.
- **fork и потоки.** После `fork` в многопоточном процессе безопасны только async-signal-safe вызовы до `exec`.
- **join и утечки.** Неприсоединённый `std::thread` при уничтожении вызывает `std::terminate`.
- **Приоритеты и планировщик.** Потоки конкурируют внутри процесса; fair scheduling не гарантирован.
- **Копирование при fork.** Copy-on-write спасает память, но дескрипторы и блокировки наследуются в хрупком состоянии.
