# Плюсы и минусы Angular для enterprise

## Суть

Angular это opinionated platform от Google: modules, DI, router, forms, HTTP client и CLI входят в коробку. TypeScript обязателен, conventions едины для больших команд. Цена: крутая кривая обучения, тяжелее bundle и миграции между major версиями.

## Что ответить на собесе

- Плюсы: структура из коробки, RxJS streams, встроенные тестовые инструменты, долгий LTS цикл.
- Минусы: boilerplate, change detection нюансы без OnPush, размер initial payload.
- Для банков и regulated доменов Angular часто выбирают за предсказуемость архитектуры.
- Честно: для маленького MVP React/Vue быстрее стартуют; Angular окупается на 50+ разработчиков и 5+ лет жизни продукта.
- Ivy и standalone components упростили tree shaking, но ментальная модель DI остаётся сложной.

## Как это работает

```ts
import { Component, inject } from '@angular/core';
import { HttpClient } from '@angular/common/http';

@Component({
  selector: 'app-greeting',
  standalone: true,
  template: `<h1>{{ headline }}</h1>`,
  styles: [`h1 { font-family: system-ui, sans-serif; }`],
})
export class GreetingComponent {
  private readonly http = inject(HttpClient);
  headline = 'Hello, Angular';

  ngOnInit() {
    this.http.get<{ title: string }>('/api/title').subscribe((res) => {
      this.headline = res.title;
    });
  }
}
```

Модульность: feature modules группируют components, pipes, directives. DI container резолвит сервисы по токенам (`providedIn: 'root'` singleton). Two-way binding `[(ngModel)]` связывает template и class, но в крупных формах предпочитают reactive forms.

CLI (`ng generate`, `ng build`, `ng test`) стандартизирует scaffolding. Jasmine + Karma (или Jest) покрывают unit, Protractor/Cypress e2e. Major upgrades идут через `ng update` с schematic migrations.

## Подводные камни

- **Default change detection.** Без OnPush лишние digest циклы бьют по performance в больших таблицах.
- **RxJS overuse.** Потоки на каждый клик усложняют отладку; иногда достаточно async pipe + signal.
- **Bundle weight.** Полный platform тяжелее library-first подхода; lazy routes обязательны.
- **Breaking migrations.** Angular 2→15 история пугает заказчиков; планируй cadence обновлений заранее.
- **Hiring pool.** Меньше кандидатов чем React, выше порог входа для джунов.
