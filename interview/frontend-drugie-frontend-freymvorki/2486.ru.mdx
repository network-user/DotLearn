# Архитектура Angular-приложения

## Суть

Angular app это дерево components с templates, orchestrated modules (legacy) или standalone APIs, services с DI, directives/pipes. Router lazy-loads feature areas. RxJS streams часто glue async data. CLI генерирует consistent structure для enterprise teams.

## Что ответить на собесе

- Component = template + class + styles; `@Component` metadata.
- Services `@Injectable` hold business logic; DI injects into constructors.
- Feature modules (or standalone routes) isolate domains: auth, billing.
- Честно: NgModules постепенно заменяются standalone components в v17+.
- Strict TS, eslint, schematics держат monorepo Nx-friendly layout.

## Как это работает

```ts
import { Component, inject } from '@angular/core';
import { CatalogService } from './catalog.service';

@Component({
  selector: 'app-product-card',
  standalone: true,
  template: `
    <h2>{{ item.title }}</h2>
    <p>{{ item.price | currency }}</p>
    <button type="button" (click)="add()">Add</button>
  `,
})
export class ProductCardComponent {
  private catalog = inject(CatalogService);
  item = { title: 'Keyboard', price: 120 };

  add() {
    this.catalog.add(this.item);
  }
}
```

Bootstrap: `main.ts` → `bootstrapApplication(AppComponent, { providers: [...] })`. Router maps paths to lazy `loadComponent`. HTTP via `HttpClient` + interceptors (auth header, error map).

Layers: smart container components fetch, dumb presentational receive `@Input()` emit `@Output()`.

## Подводные камни

- **God modules.** Everything imported in AppModule → huge initial bundle.
- **Shared mutable service state.** Race without observables discipline.
- **Template-driven forms at scale.** Reactive forms + typed FormGroup safer.
- **Zone.js overhead.** Heavy third-party libs trigger extra CD cycles.
- **Upgrade lag.** Enterprise stuck two majors behind; migration cost real.
