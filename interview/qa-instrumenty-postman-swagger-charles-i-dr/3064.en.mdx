# How do you substitute an outgoing request in Charles with Breakpoints?

## In short

A breakpoint pauses the request (and optionally the response) before it hits the server. QA edits URL, headers, or JSON body, then Execute. This validates server validation, RBAC, and injection handling without changing the client build.

## How to answer in an interview

- Proxy → Breakpoint Settings → enable Request breakpoint for host/path.
- Repro action → Charles Breakpoint window → Edit Request → Execute.
- Change amount to negative, swap role header, use expired token.
- Repeat for a matrix of negative cases.
- Honestly: breakpoints are manual; regression suites belong in automated API tests.

## How it works

```gherkin
Feature: Charles request substitution

  Scenario: Send invalid quantity to cart API
    Given breakpoint on POST /api/v2/cart/items
    When original body qty 2 intercepted
    And tester edits qty to -1
    And Execute forwards to server
    Then response 400 and UI shows validation message

  Scenario: Test admin endpoint as regular user
    Given breakpoint on GET /admin/users
    When Authorization header replaced with user token
    Then server returns 403 and app handles forbidden state

  Scenario: Omit required header
    Given breakpoint removes Idempotency-Key header
    When POST /payments executed
    Then 409 duplicate charge prevented or error surfaced QA verifies policy
```

Original and edited body (logical form):

```json
{"sku":"WB-991","qty":2,"warehouseId":"wh-east"}
→ {"sku":"WB-991","qty":-1,"warehouseId":"wh-east"}
```

Alternatives: Rewrite for permanent header inject; Edit & Repeat on a captured request without breakpoint next time.

## Pitfalls

- **Breakpoint on all URLs.** Every app tap hangs; narrow the rule.
- **Edit wrong tab (Response).** Request unchanged; triage confusion.
- **Invalid JSON after pretty edit.** Trailing comma → opaque 400.
- **Idempotency:** repeat Execute creates duplicate side effect on staging.
- **HTTPS host not in SSL list.** Breakpoint never fires; tunnel only.
