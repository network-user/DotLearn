# Form by reference or by object: when to use which

## In short

Open a form by infobase reference when editing an existing record, or with a new object/parameters when creating a document or a common form not tied to a specific reference. The choice affects `Объект` fill, locks, and reopen by uniqueness.

## How to answer in an interview

- I pass `Ключ` in parameters for an existing item and an empty/new `Объект` for creation.
- In an interview I show `ОткрытьФорму` with a reference parameter vs `СоздатьДокумент` plus object form.
- I stress list forms work with dynamic lists, not `ДокументОбъект`.
- I honestly note opening by reference without edit rights yields read-only when roles are set correctly.
- Common forms (report, wizard) open without `Ключ`, only a parameter structure.

## How it works

Existing counterparty by reference:

```bsl
&НаКлиенте
Процедура ОткрытьКонтрагента(СсылкаКонтрагента)
    ПараметрыФормы = Новый Структура("Ключ", СсылкаКонтрагента);
    ОткрытьФорму("Справочник.Контрагенты.ФормаЭлемента", ПараметрыФормы);
КонецПроцедуры
```

New document via object:

```bsl
&НаСервере
Функция СоздатьРеализациюНаСервере()
    НовыйДокумент = Документы.РеализацияТоваров.СоздатьДокумент();
    НовыйДокумент.Дата = ТекущаяДатаСеанса();
    Возврат НовыйДокумент;
КонецФункции

&НаКлиенте
Процедура СоздатьРеализацию(Команда)
    ПараметрыФормы = Новый Структура("Объект", СоздатьРеализациюНаСервере());
    ОткрытьФорму("Документ.РеализацияТоваров.ФормаДокумента", ПараметрыФормы);
КонецПроцедуры
```

Common form without a metadata object:

```bsl
&НаКлиенте
Процедура ОткрытьМастерИмпорта()
    ОткрытьФорму("ОбщаяФорма.МастерИмпорта");
КонецПроцедуры
```

## Pitfalls

- **Key vs Object.** Parameter confusion: form opens empty or with the wrong record.
- **Unsaved object.** Passed `Объект` without reference, forgot to write before reopen by UUID.
- **Locking.** Two users on one `Ключ` conflict on write, not on open.
- **Rights.** Reference to a forbidden area may list but form open can fail.
- **Window reuse.** `Уникальность` by reference fails if opened without key parameter.
