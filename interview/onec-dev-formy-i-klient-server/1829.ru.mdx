# Форма по ссылке или по объекту: когда что

## Суть

Форму открывают по ссылке на элемент ИБ, когда редактируют существующую запись, или с новым объектом/параметрами, когда создают документ или общую форму без привязки к конкретной ссылке. Выбор влияет на автозаполнение `Объект`, блокировки и повторное открытие по уникальности.

## Что ответить на собесе

- Я передаю `Ключ` в параметрах для существующего элемента и пустой `Объект` для нового.
- На собеседовании показываю `ОткрытьФорму` с параметром ссылки vs `СоздатьДокумент` + форма объекта.
- Подчёркиваю, что форма списка работает с динамическим списком, не с `ДокументОбъект`.
- Честно уточняю: открытие по ссылке без прав на изменение даёт форму только для просмотра при корректных ролях.
- Общие формы (отчёт, мастер) открывают без `Ключ`, только со `Структура` параметров.

## Как это работает

Существующий контрагент по ссылке:

```bsl
&НаКлиенте
Процедура ОткрытьКонтрагента(СсылкаКонтрагента)
    ПараметрыФормы = Новый Структура("Ключ", СсылкаКонтрагента);
    ОткрытьФорму("Справочник.Контрагенты.ФормаЭлемента", ПараметрыФормы);
КонецПроцедуры
```

Новый документ через объект:

```bsl
&НаСервере
Функция СоздатьРеализациюНаСервере()
    НовыйДокумент = Документы.РеализацияТоваров.СоздатьДокумент();
    НовыйДокумент.Дата = ТекущаяДатаСеанса();
    Возврат НовыйДокумент;
КонецФункции

&НаКлиенте
Процедура СоздатьРеализацию(Команда)
    ПараметрыФормы = Новый Структура("Объект", СоздатьРеализациюНаСервере());
    ОткрытьФорму("Документ.РеализацияТоваров.ФормаДокумента", ПараметрыФормы);
КонецПроцедуры
```

Общая форма без объекта метаданных:

```bsl
&НаКлиенте
Процедура ОткрытьМастерИмпорта()
    ОткрытьФорму("ОбщаяФорма.МастерИмпорта");
КонецПроцедуры
```

## Подводные камни

- **Ключ vs Объект.** Путаница параметров: форма открывается пустой или с неверной записью.
- **Не записанный объект.** Передали `Объект` без ссылки, забыли записать перед повторным открытием по UUID.
- **Блокировка.** Два пользователя по одному `Ключ` получают конфликт при записи, не при открытии.
- **Права.** Ссылка на чужой раздел без RLS видна в списке, но форма может отказать в открытии.
- **Повторное использование окна.** `Уникальность` по ссылке не сработает, если открыть без параметра ключа.
