# &AtClient, &AtServer, and &AtServerNoContext directives

## In short

Compilation directives mark which side runs a form or common module procedure. `&НаКлиенте` handles UI, `&НаСервере` has form context and infobase access, `&НаСервереБезКонтекста` runs on the server in isolation without form data binding.

## How to answer in an interview

- I keep a thin client: UI and input validation on the client, queries and writes on the server.
- In an interview I sketch the chain: client button handler calls a server function with parameters.
- I stress server code in a form module does not ship to the client binary when marked correctly.
- I honestly note extra `&НаСервере` inside loops cause excessive server calls.
- I use `&НаСервереБезКонтекста` for reusable functions without access to form `Объект`.

## How it works

Client shows a message:

```bsl
&НаКлиенте
Процедура ПоказатьПодсказку(Команда)
    ПоказатьПредупреждение(, "Действие выполнено на клиенте");
КонецПроцедуры
```

Server reads data:

```bsl
&НаСервере
Функция ПолучитьОстатокНоменклатуры(Номенклатура)
    Запрос = Новый Запрос;
    Запрос.Текст =
    "ВЫБРАТЬ СУММА(Остаток) КАК Остаток
    |ИЗ РегистрНакопления.ТоварыНаСкладах.Остатки
    |ГДЕ Номенклатура = &Номенклатура";
    Запрос.УстановитьПараметр("Номенклатура", Номенклатура);
    Выборка = Запрос.Выполнить().Выбрать();
    Возврат ?(Выборка.Следующий(), Выборка.Остаток, 0);
КонецФункции
```

No context for a utility:

```bsl
&НаСервереБезКонтекста
Функция НормализоватьСтроку(ИсходнаяСтрока)
    Возврат СокрЛП(ВРег(ИсходнаяСтрока));
КонецФункции
```

## Pitfalls

- **Queries on client.** Direct query in `&НаКлиенте` is unavailable; use a server call.
- **Large value table transfer.** Each call serializes data; batch payloads.
- **Mixed contexts.** Calling `&НаСервере` from `&НаСервереБезКонтекста` without form parameters cannot see `Объект`.
- **Web client.** Some client APIs are unavailable; check compatibility.
- **Debugging.** Breakpoints in server methods require attaching to the server process.
