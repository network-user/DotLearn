# Из чего состоит IP

## 1️⃣ Как кратко ответить

IP (Internet Protocol) состоит из двух основных версий: IPv4 и IPv6. IPv4 адреса состоят из 32 бит и обычно записываются в виде четырех десятичных чисел, разделенных точками (например, 192.168.1.1). IPv6 адреса состоят из 128 бит и записываются в виде восьми групп шестнадцатеричных чисел, разделенных двоеточиями (например, 2001:0db8:85a3:0000:0000:8a2e:0370:7334).

## 2️⃣ Подробное объяснение темы

IP (Internet Protocol) — это основной протокол, который используется для передачи данных в сети Интернет. Он определяет, как данные должны быть упакованы, адресованы, переданы, маршрутизированы и получены на конечных узлах. Существует две основные версии IP: IPv4 и IPv6.

**IPv4:**

IPv4 — это четвертая версия IP, которая была разработана в 1980-х годах. Она использует 32-битные адреса, что позволяет создать около 4,3 миллиарда уникальных адресов. Формат IPv4 адреса — это четыре десятичных числа, каждое из которых может принимать значение от 0 до 255, разделенные точками. Например, 192.168.1.1.

- **192**: Первое число в адресе, представляющее старший байт.
- **168**: Второе число, представляющее следующий байт.
- **1**: Третье число, представляющее третий байт.
- **1**: Четвертое число, представляющее младший байт.

Каждое число в адресе IPv4 представляет собой один байт (8 бит), и все четыре числа вместе составляют 32 бита.

**IPv6:**

IPv6 был разработан для решения проблемы исчерпания адресов IPv4. Он использует 128-битные адреса, что позволяет создать огромное количество уникальных адресов (примерно 3.4×10^38). Формат IPv6 адреса — это восемь групп шестнадцатеричных чисел, разделенных двоеточиями. Например, 2001:0db8:85a3:0000:0000:8a2e:0370:7334.

- **2001**: Первая группа, представляющая старший сегмент адреса.
- **0db8**: Вторая группа, следующая за первой.
- **85a3**: Третья группа.
- **0000**: Четвертая группа, может быть сокращена до **::** если это последовательность нулей.
- **0000**: Пятая группа, также может быть сокращена.
- **8a2e**: Шестая группа.
- **0370**: Седьмая группа.
- **7334**: Восьмая группа, представляющая младший сегмент адреса.

IPv6 адреса могут содержать сокращения, чтобы упростить запись. Например, последовательности нулей могут быть заменены на двойное двоеточие (::), но это сокращение может быть использовано только один раз в адресе.

**Применение и важность:**

IP адресация необходима для идентификации устройств в сети и обеспечения их взаимодействия. Без IP адресов невозможно было бы направлять данные от одного устройства к другому в глобальной сети. IPv4 все еще широко используется, но из-за ограниченного количества адресов происходит постепенный переход на IPv6, который обеспечивает более масштабируемую и гибкую адресацию.
