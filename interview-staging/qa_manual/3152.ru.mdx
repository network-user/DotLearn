# Какие знаешь теги в HTML

## 1️⃣ Как кратко ответить

HTML теги — это элементы, которые используются для создания структуры веб-страницы. Основные теги включают: ``, ``, ``, ``, ``-``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``, ``.

## 2️⃣ Подробное объяснение темы

HTML (HyperText Markup Language) — это язык разметки, используемый для создания и структурирования веб-страниц. HTML состоит из элементов, которые обозначаются тегами. Теги определяют, как браузер должен отображать содержимое страницы. Каждый тег имеет свое назначение и используется для различных целей.

### Основные теги HTML

1. **``**: Корневой элемент HTML-документа. Все остальные элементы должны находиться внутри этого тега.
2. **``**: Содержит метаинформацию о документе, такую как заголовок, ссылки на стили и скрипты.
3. **``**: Определяет заголовок документа, который отображается в заголовке окна браузера или на вкладке.
4. **``**: Содержит основное содержимое HTML-документа, которое отображается на веб-странице.
5. **``-``**: Заголовки, которые используются для обозначения заголовков разного уровня. `` — самый важный заголовок, `` — наименее важный.
6. **``**: Определяет абзац текста.
7. **``**: Создает гиперссылку. Атрибут `href` указывает URL, на который ведет ссылка.
8. **``**: Вставляет изображение. Атрибуты `src` и `alt` указывают путь к изображению и альтернативный текст соответственно.
9. **`` и ``**: Создают неупорядоченные (списки с маркерами) и упорядоченные (нумерованные списки) списки соответственно.
10. **``**: Элемент списка, используется внутри `` или ``.
11. **``**: Блочный элемент, используемый для группировки других элементов и применения стилей.
12. **``**: Строчный элемент, используемый для стилизации части текста или других элементов.
13. **``, ``, ``, ``**: Используются для создания таблиц. `` — контейнер для таблицы, `` — строка таблицы, `` — ячейка, `` — заголовочная ячейка.
14. **``**: Создает форму для ввода данных пользователем.
15. **``**: Поле ввода данных в форме. Атрибут `type` определяет тип ввода (например, текст, пароль, кнопка).
16. **``**: Создает кнопку, которую можно нажать для выполнения действия.
17. **``**: Используется для подключения внешних ресурсов, таких как таблицы стилей.
18. **``**: Определяет метаинформацию о документе, такую как кодировка, автор и описание.
19. **``**: Включает или ссылается на скрипты, такие как JavaScript.
20. **``**: Включает CSS-стили, которые применяются к элементам HTML.

### Пример использования тегов

```html
<!DOCTYPE html>
<html>
<head>
    <title>Пример HTML-документа</title>
    <meta charset="UTF-8">
    <link rel="stylesheet" href="styles.css">
</head>
<body>
    <h1>Заголовок страницы</h1>
    <p>Это пример абзаца текста. <a href="https://example.com">Ссылка на сайт</a>.</p>
    <img src="image.jpg" alt="Описание изображения">
    <ul>
        <li>Элемент списка 1</li>
        <li>Элемент списка 2</li>
    </ul>
    <form action="/submit" method="post">
        <input type="text" name="username" placeholder="Введите имя">
        <button type="submit">Отправить</button>
    </form>
</body>
</html>
```

- ``: Объявление типа документа, указывающее, что используется HTML5.
- ``: Начало HTML-документа.
- ``: Содержит метаинформацию, такую как ``, ``, и ``.
- ``: Основное содержимое страницы, включая заголовки, абзацы, изображения, списки и формы.
- ``: Заголовок первого уровня.
- ``: Абзац текста с вложенной ссылкой ``.
- ``: Изображение с атрибутами `src` и `alt`.
- `` и ``: Неупорядоченный список с элементами.
- ``, ``, ``: Форма с полем ввода и кнопкой отправки.
