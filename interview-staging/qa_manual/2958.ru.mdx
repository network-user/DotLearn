# В чём разница между IP и MAC адрес

## 1️⃣ Как кратко ответить

IP-адрес — это логический адрес, используемый для идентификации устройства в сети и маршрутизации данных между сетями. MAC-адрес — это физический адрес, уникально идентифицирующий сетевое устройство на уровне канала передачи данных в пределах одной сети.

## 2️⃣ Подробное объяснение темы

IP-адрес и MAC-адрес — это два различных типа адресов, которые используются для идентификации устройств в сетях, но они работают на разных уровнях модели OSI и выполняют разные функции.

**IP-адрес (Internet Protocol address):**

1. **Назначение и уровень модели OSI:**

IP-адрес работает на сетевом уровне (уровень 3) модели OSI.
2. Он используется для логической идентификации устройства в сети и маршрутизации данных между различными сетями.
3. **Формат и типы:**

IPv4-адрес состоит из 32 бит и обычно записывается в виде четырех десятичных чисел, разделенных точками (например, 192.168.1.1).
4. IPv6-адрес состоит из 128 бит и записывается в виде восьми групп шестнадцатеричных чисел, разделенных двоеточиями (например, 2001:0db8:85a3:0000:0000:8a2e:0370:7334).
5. **Динамическое и статическое назначение:**

IP-адреса могут быть статическими (постоянно назначенными) или динамическими (назначаемыми автоматически с помощью DHCP).
6. **Маршрутизация:**

IP-адреса используются для маршрутизации пакетов данных между различными сетями, что позволяет устройствам обмениваться данными через интернет.

**MAC-адрес (Media Access Control address):**

1. **Назначение и уровень модели OSI:**

MAC-адрес работает на канальном уровне (уровень 2) модели OSI.
2. Он используется для уникальной идентификации сетевого устройства в пределах одной локальной сети (LAN).
3. **Формат:**

MAC-адрес состоит из 48 бит и обычно записывается в виде шести пар шестнадцатеричных чисел, разделенных двоеточиями или дефисами (например, 00:1A:2B:3C:4D:5E).
4. **Уникальность и назначение:**

MAC-адреса назначаются производителями сетевых устройств и являются уникальными для каждого устройства.
5. **Локальная передача данных:**

MAC-адреса используются для передачи данных внутри одной локальной сети, обеспечивая доставку пакетов данных к правильному устройству.

**Пример использования:**

Когда устройство отправляет данные через интернет, сначала используется MAC-адрес для передачи данных внутри локальной сети до маршрутизатора. Затем маршрутизатор использует IP-адрес для отправки данных через интернет к целевой сети. В целевой сети данные снова передаются с использованием MAC-адреса до конечного устройства.

Таким образом, IP-адреса и MAC-адреса работают совместно, обеспечивая эффективную передачу данных как внутри локальных сетей, так и между различными сетями.
