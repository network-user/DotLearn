# Что такое HTML

## 1️⃣ Как кратко ответить

HTML (HyperText Markup Language) — это стандартный язык разметки для создания и структурирования веб-страниц. Он используется для определения структуры и содержания веб-документов, таких как текст, изображения, ссылки и другие элементы.

## 2️⃣ Подробное объяснение темы

HTML, или HyperText Markup Language, является основой для создания веб-страниц и веб-приложений. Он определяет структуру веб-документа с помощью различных тегов и атрибутов, которые указывают браузеру, как отображать содержимое.

### Основные элементы HTML

1. **Теги**: Основные строительные блоки HTML. Они заключены в угловые скобки, например, ``. Теги обычно парные: открывающий `` и закрывающий ``, за исключением некоторых одиночных тегов, таких как ``.
2. **Атрибуты**: Дополнительные параметры, которые предоставляют дополнительную информацию о теге. Они указываются внутри открывающего тега, например, ``.
3. **Элементы**: Комбинация тегов и содержимого между ними. Например, `This is a paragraph.` — это элемент.

### Пример HTML-документа

```html
<!DOCTYPE html>
<html>
<head>
    <title>Пример HTML-документа</title>
</head>
<body>
    <h1>Заголовок страницы</h1>
    <p>Это пример параграфа текста на веб-странице.</p>
    <a href="https://example.com">Ссылка на другой сайт</a>
    <img src="image.jpg" alt="Описание изображения">
</body>
</html>
```

### Объяснение кода

- ``: Объявление типа документа, указывающее браузеру, что используется HTML5.
- ``: Корневой элемент, содержащий весь HTML-документ.
- ``: Секция, содержащая метаданные о документе, такие как ``, который задает заголовок страницы в браузере.
- ``: Основная часть документа, где размещается видимое содержимое.
- ``: Тег заголовка первого уровня, используемый для обозначения основного заголовка страницы.
- ``: Тег параграфа, используемый для отображения текста.
- ``: Тег ссылки, с атрибутом `href`, указывающим URL, на который ведет ссылка.
- ``: Тег изображения, с атрибутами `src` для указания пути к изображению и `alt` для альтернативного текста.

### Зачем нужен HTML

HTML необходим для создания структуры веб-страниц. Он позволяет разработчикам определять, как будет организовано содержимое, и как оно будет взаимодействовать с пользователем. HTML тесно интегрируется с CSS (Cascading Style Sheets) для стилизации и JavaScript для добавления интерактивности, что делает его неотъемлемой частью веб-разработки.
