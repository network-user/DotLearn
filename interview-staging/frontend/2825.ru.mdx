# Как получить тип возвращаемого значения из функции

## 1️⃣ Как кратко ответить

В TypeScript для получения типа возвращаемого значения из функции используется оператор `ReturnType`. Он извлекает тип, который функция возвращает, и позволяет использовать его в других частях кода.

## 2️⃣ Подробное объяснение темы

В TypeScript, языке программирования, который добавляет статическую типизацию к JavaScript, часто возникает необходимость определить тип возвращаемого значения функции. Это может быть полезно для обеспечения согласованности типов в коде и для улучшения читаемости и поддержки кода.

TypeScript предоставляет встроенный условный тип `ReturnType`, который позволяет извлечь тип возвращаемого значения из функции. Это особенно полезно, когда функция сложная, и вы хотите использовать её возвращаемый тип в других частях вашего приложения.

### Пример использования `ReturnType`

Рассмотрим простой пример функции, которая возвращает строку:

```typescript
function greet(name: string): string {
    return `Hello, ${name}!`;
}
```

Функция `greet` принимает строку `name` и возвращает строку приветствия. Чтобы получить тип возвращаемого значения этой функции, мы можем использовать `ReturnType`:

```typescript
type GreetReturnType = ReturnType<typeof greet>;
```

Здесь `GreetReturnType` будет иметь тип `string`, так как функция `greet` возвращает строку.

### Как это работает

- **`typeof greet`**: Оператор `typeof` используется для получения типа функции `greet`. В TypeScript `typeof` может использоваться для получения типа переменной или функции.
- **`ReturnType`**: `ReturnType` — это условный тип, который принимает тип функции и извлекает из него тип возвращаемого значения. В данном случае, он извлекает тип `string` из функции `greet`.

### Зачем это нужно

Использование `ReturnType` позволяет:

1. **Уменьшить дублирование кода**: Вместо того чтобы вручную указывать тип возвращаемого значения в нескольких местах, вы можете извлечь его из функции и использовать везде, где это необходимо.
2. **Обеспечить согласованность типов**: Если тип возвращаемого значения функции изменится, `ReturnType` автоматически обновит все места, где он используется, что снижает вероятность ошибок.
3. **Улучшить читаемость и поддержку кода**: Явное указание типов делает код более понятным и легким для поддержки.

### Пример с более сложной функцией

Рассмотрим более сложный пример, где функция возвращает объект:

```typescript
function getUser(id: number) {
    return {
        id,
        name: `User${id}`,
        isActive: true
    };
}
```

Функция `getUser` возвращает объект с тремя свойствами: `id`, `name` и `isActive`. Чтобы получить тип этого объекта, используем `ReturnType`:

```typescript
type UserType = ReturnType<typeof getUser>;
```

Теперь `UserType` будет иметь следующий тип:

```typescript
{
    id: number;
    name: string;
    isActive: boolean;
}
```

Это позволяет использовать `UserType` в других частях кода, например, для аннотации типов переменных или параметров функций, что делает код более надежным и легким для понимания.
