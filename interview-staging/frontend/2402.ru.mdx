# Как сделать, чтобы ссылка открывалась в новом окне

## 1️⃣ Как кратко ответить

Для того чтобы ссылка открывалась в новом окне, необходимо использовать атрибут `target="_blank"` в теге ``.

## 2️⃣ Подробное объяснение темы

В HTML, чтобы сделать так, чтобы ссылка открывалась в новом окне или вкладке браузера, используется атрибут `target` в теге ``. Значение этого атрибута определяет, где будет открываться связанный документ.

### Зачем это нужно

Открытие ссылки в новом окне может быть полезно, когда вы хотите, чтобы пользователи оставались на текущей странице, но при этом могли просмотреть дополнительную информацию или перейти на другой ресурс. Это особенно актуально для внешних ссылок, чтобы не прерывать взаимодействие пользователя с вашим сайтом.

### Как это работает

Атрибут `target` может принимать несколько значений, но для открытия в новом окне используется значение `_blank`. Это значение указывает браузеру открыть ссылку в новой вкладке или окне, в зависимости от настроек браузера пользователя.

### Пример кода

```html
<a href="https://www.example.com" target="_blank">Посетить Example</a>
```

### Объяснение кода:

- ``: Это HTML-тег, который используется для создания гиперссылок.
- `href="https://www.example.com"`: Атрибут `href` указывает URL-адрес, на который будет вести ссылка. В данном случае, это "https://www.example.com".
- `target="_blank"`: Атрибут `target` с значением `_blank` указывает, что ссылка должна открываться в новом окне или вкладке.
- `Посетить Example`: Текст, который будет отображаться пользователю как кликабельная ссылка.

### Важные моменты

1. **Безопасность**: Использование `target="_blank"` может представлять риск безопасности, известный как "tabnabbing". Это когда новая вкладка может изменить оригинальную страницу через JavaScript. Чтобы предотвратить это, рекомендуется использовать атрибут `rel="noopener noreferrer"`.
```html
<a href="https://www.example.com" target="_blank" rel="noopener noreferrer">Посетить Example</a>
```

`rel="noopener noreferrer"`: Этот атрибут предотвращает доступ новой вкладки к объекту `window.opener`, что защищает оригинальную страницу от потенциальных атак.
2. **Пользовательский опыт**: Не злоупотребляйте открытием ссылок в новых окнах, так как это может раздражать пользователей. Лучше всего использовать это для внешних ссылок или в случаях, когда это действительно необходимо для удобства пользователя.

Таким образом, использование `target="_blank"` — это простой и эффективный способ контролировать поведение ссылок на вашем сайте, обеспечивая при этом безопасность и удобство для пользователей.
