# Какие методы массива не мутируют исходный массив

## 1️⃣ Как кратко ответить

Методы массива, которые не мутируют исходный массив, включают `map()`, `filter()`, `reduce()`, `concat()`, `slice()`, `join()`, `includes()`, `indexOf()`, `lastIndexOf()`, `find()`, `findIndex()`, `every()`, и `some()`. Эти методы возвращают новый массив или значение, не изменяя оригинальный массив.

## 2️⃣ Подробное объяснение темы

В JavaScript массивы являются изменяемыми объектами, что означает, что их содержимое может быть изменено. Однако, не все методы массивов изменяют (мутируют) исходный массив. Некоторые методы возвращают новый массив или значение, оставляя оригинальный массив неизменным. Это важно для сохранения данных в их первоначальном виде и предотвращения непреднамеренных изменений.

**Методы, которые не мутируют массив:**

1. **`map()`**: Создает новый массив с результатами вызова предоставленной функции для каждого элемента в массиве.
```javascript
const numbers = [1, 2, 3];
const doubled = numbers.map(num => num * 2);
// numbers: [1, 2, 3]
// doubled: [2, 4, 6]
```

`numbers` остается неизменным.
2. `doubled` — новый массив с удвоенными значениями.
3. **`filter()`**: Создает новый массив со всеми элементами, прошедшими проверку, реализованную в предоставленной функции.
```javascript
const numbers = [1, 2, 3, 4];
const evenNumbers = numbers.filter(num => num % 2 === 0);
// numbers: [1, 2, 3, 4]
// evenNumbers: [2, 4]
```

`numbers` остается неизменным.
4. `evenNumbers` — новый массив с четными числами.
5. **`reduce()`**: Применяет функцию к аккумулятору и каждому значению массива (слева направо), сводя его к единственному значению.
```javascript
const numbers = [1, 2, 3, 4];
const sum = numbers.reduce((acc, num) => acc + num, 0);
// numbers: [1, 2, 3, 4]
// sum: 10
```

`numbers` остается неизменным.
6. `sum` — сумма всех элементов массива.
7. **`concat()`**: Возвращает новый массив, состоящий из массива, на котором он был вызван, соединенного с другими массивами и/или значениями.
```javascript
const array1 = [1, 2];
const array2 = [3, 4];
const combined = array1.concat(array2);
// array1: [1, 2]
// array2: [3, 4]
// combined: [1, 2, 3, 4]
```

`array1` и `array2` остаются неизменными.
8. `combined` — новый массив, объединяющий `array1` и `array2`.
9. **`slice()`**: Возвращает новый массив, содержащий копию части исходного массива.
```javascript
const numbers = [1, 2, 3, 4];
const sliced = numbers.slice(1, 3);
// numbers: [1, 2, 3, 4]
// sliced: [2, 3]
```

`numbers` остается неизменным.
10. `sliced` — новый массив, содержащий элементы с индексами от 1 до 2.
11. **`join()`**: Объединяет все элементы массива в строку.
```javascript
const elements = ['Fire', 'Air', 'Water'];
const joined = elements.join('-');
// elements: ['Fire', 'Air', 'Water']
// joined: 'Fire-Air-Water'
```

`elements` остается неизменным.
12. `joined` — строка, содержащая элементы массива, разделенные дефисом.
13. **`includes()`**: Определяет, содержит ли массив определенное значение, возвращая `true` или `false`.
```javascript
const numbers = [1, 2, 3];
const hasTwo = numbers.includes(2);
// numbers: [1, 2, 3]
// hasTwo: true
```

`numbers` остается неизменным.
14. `hasTwo` — булево значение, указывающее на наличие элемента.
15. **`indexOf()` и `lastIndexOf()`**: Возвращают первый или последний индекс, по которому данный элемент может быть найден в массиве, или -1, если элемент не найден.
```javascript
const numbers = [1, 2, 3, 2];
const index = numbers.indexOf(2);
const lastIndex = numbers.lastIndexOf(2);
// numbers: [1, 2, 3, 2]
// index: 1
// lastIndex: 3
```

`numbers` остается неизменным.
16. `index` и `lastIndex` — индексы первого и последнего вхождения элемента.
17. **`find()` и `findIndex()`**: Возвращают первый элемент или его индекс, который удовлетворяет предоставленной функции.
```javascript
const numbers = [1, 2, 3, 4];
const found = numbers.find(num => num > 2);
const foundIndex = numbers.findIndex(num => num > 2);
// numbers: [1, 2, 3, 4]
// found: 3
// foundIndex: 2
```

`numbers` остается неизменным.
18. `found` — первый элемент, удовлетворяющий условию.
19. `foundIndex` — индекс первого элемента, удовлетворяющего условию.
20. **`every()` и `some()`**: Проверяют, удовлетворяют ли все или некоторые элементы массива условию, заданному в функции.
```javascript
const numbers = [1, 2, 3, 4];
const allPositive = numbers.every(num => num > 0);
const someGreaterThanTwo = numbers.some(num => num > 2);
// numbers: [1, 2, 3, 4]
// allPositive: true
// someGreaterThanTwo: true
```

`numbers` остается неизменным.
21. `allPositive` и `someGreaterThanTwo` — булевые значения, указывающие на выполнение условия.

Эти методы полезны, когда необходимо работать с массивами, не изменяя их исходное состояние, что особенно важно в функциональном программировании и при работе с неизменяемыми данными.
