# Селектор *

## 1️⃣ Как кратко ответить

Селектор `*` в CSS называется универсальным селектором и применяется для выбора всех элементов на веб-странице. Он может использоваться для установки общих стилей или сброса стилей по умолчанию.

## 2️⃣ Подробное объяснение темы

Универсальный селектор `*` в CSS — это мощный инструмент, который позволяет выбрать все элементы на веб-странице. Он полезен в ситуациях, когда необходимо применить определенные стили ко всем элементам без исключения. Это может быть полезно для сброса стилей по умолчанию или для установки базовых стилей, которые будут применяться ко всем элементам.

### Применение универсального селектора

1. **Сброс стилей по умолчанию**: Браузеры имеют свои собственные стили по умолчанию для HTML-элементов. Универсальный селектор может быть использован для сброса этих стилей, чтобы обеспечить единообразие отображения в разных браузерах.
2. **Установка базовых стилей**: Универсальный селектор может применяться для задания базовых стилей, таких как шрифты, отступы и границы, которые будут применяться ко всем элементам.

### Пример использования

```css
* {
  margin: 0; /* Устанавливает отступы всех элементов в 0 */
  padding: 0; /* Устанавливает внутренние отступы всех элементов в 0 */
  box-sizing: border-box; /* Изменяет модель коробки для всех элементов, чтобы включать границы и отступы в общую ширину и высоту */
}
```

- `* { ... }`: Универсальный селектор, который выбирает все элементы на странице.
- `margin: 0;`: Устанавливает внешние отступы всех элементов в 0, что помогает устранить нежелательные отступы, которые могут быть заданы браузером по умолчанию.
- `padding: 0;`: Устанавливает внутренние отступы всех элементов в 0, что также помогает устранить нежелательные отступы.
- `box-sizing: border-box;`: Изменяет модель коробки для всех элементов, чтобы ширина и высота включали в себя границы и отступы. Это упрощает управление размерами элементов.

### Важные замечания

- Использование универсального селектора может повлиять на производительность, особенно на больших страницах, так как он применяет стили ко всем элементам.
- Следует использовать его с осторожностью и только в тех случаях, когда это действительно необходимо, чтобы избежать ненужной нагрузки на браузер.

Универсальный селектор — это инструмент, который может значительно упростить управление стилями на веб-странице, но его следует использовать осознанно и с пониманием его влияния на производительность и структуру стилей.
