# Как получить элемент страницы по селектору

## 1️⃣ Как кратко ответить

Для получения элемента страницы по селектору в JavaScript используется метод `document.querySelector()`. Он возвращает первый элемент, соответствующий указанному CSS-селектору. Если элемент не найден, возвращается `null`.

## 2️⃣ Подробное объяснение темы

В веб-разработке часто возникает необходимость взаимодействовать с элементами HTML-документа. Для этого используется объект `document`, который предоставляет доступ к DOM (Document Object Model) страницы. Один из самых распространенных способов получить элемент — это использовать метод `document.querySelector()`.

### Что такое `document.querySelector()`?

`document.querySelector()` — это метод, который позволяет найти первый элемент в документе, соответствующий заданному CSS-селектору. Селектор может быть любым валидным CSS-селектором, например, селектор класса, идентификатора или тега.

### Пример использования

Рассмотрим пример, где мы хотим получить элемент с классом `button` и изменить его текст.

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
</head>
<body>
    <button class="button">Click me</button>

    <script>
        // Получаем элемент с классом 'button'
        const buttonElement = document.querySelector('.button');

        // Проверяем, что элемент найден
        if (buttonElement) {
            // Изменяем текст элемента
            buttonElement.textContent = 'Button Clicked';
        }
    </script>
</body>
</html>
```

### Пояснение к коду

1. **HTML-разметка**: Создаем простую HTML-страницу с кнопкой, у которой есть класс `button`.
2. **Получение элемента**:
```javascript
const buttonElement = document.querySelector('.button');
```
Здесь мы используем `document.querySelector('.button')`, чтобы найти первый элемент с классом `button`. Метод возвращает элемент, если он найден, или `null`, если элемента с таким селектором нет.
3. **Проверка на существование**:
```javascript
if (buttonElement) {
```
Проверяем, что элемент действительно найден, чтобы избежать ошибок при попытке доступа к свойствам `null`.
4. **Изменение текста**:
```javascript
buttonElement.textContent = 'Button Clicked';
```
Если элемент найден, изменяем его текстовое содержимое с помощью свойства `textContent`.

### Зачем это нужно?

Использование `document.querySelector()` позволяет легко и быстро находить элементы на странице для их последующего изменения или манипуляции. Это особенно полезно для динамического изменения контента, обработки событий и создания интерактивных пользовательских интерфейсов.

### Где применяется?

`document.querySelector()` широко используется в веб-разработке для:

- Изменения стилей элементов.
- Обработки событий, например, кликов по кнопкам.
- Динамического изменения содержимого страницы.
- Валидации форм и получения данных из них.

Этот метод является мощным инструментом для работы с DOM и позволяет разработчикам создавать более интерактивные и отзывчивые веб-приложения.
