# Типы данных в JavaScript

## 1️⃣ Как кратко ответить

В JavaScript существует семь примитивных типов данных: `number`, `string`, `boolean`, `null`, `undefined`, `symbol`, `bigint`, и один не примитивный тип — `object`. Примитивные типы неизменяемы, а объекты могут содержать коллекции значений и более сложные сущности.

## 2️⃣ Подробное объяснение темы

JavaScript — это язык программирования, который поддерживает несколько типов данных. Понимание этих типов данных важно для эффективного написания кода и предотвращения ошибок. В JavaScript есть семь примитивных типов данных и один не примитивный тип.

### Примитивные типы данных

1. **Number**

Представляет числовые значения. Включает как целые числа, так и числа с плавающей точкой.
2. Пример:```javascript
let age = 30; // целое число
let price = 19.99; // число с плавающей точкой
```
3. **String**

Представляет текстовые данные. Строки заключаются в одинарные, двойные или обратные кавычки.
4. Пример:```javascript
let name = "Alice";
let greeting = 'Hello, world!';
```
5. **Boolean**

Представляет логические значения: `true` или `false`.
6. Пример:```javascript
let isActive = true;
let isComplete = false;
```
7. **Null**

Специальное значение, представляющее "ничего" или "пустое значение".
8. Пример:```javascript
let emptyValue = null;
```
9. **Undefined**

Значение переменной, которая была объявлена, но не инициализирована.
10. Пример:```javascript
let notAssigned;
console.log(notAssigned); // undefined
```
11. **Symbol**

Уникальные и неизменяемые идентификаторы. Используются для создания уникальных свойств объектов.
12. Пример:```javascript
let uniqueId = Symbol('id');
```
13. **BigInt**

Представляет целые числа произвольной длины. Используется для работы с числами, которые превышают безопасный диапазон `Number`.
14. Пример:```javascript
let bigNumber = BigInt(123456789012345678901234567890);
```

### Не примитивный тип данных

1. **Object**

Коллекция свойств, где каждое свойство состоит из ключа и значения. Объекты могут содержать другие объекты, массивы и функции.
2. Пример:```javascript
let person = {
  name: "Bob",
  age: 25,
  greet: function() {
    console.log("Hello!");
  }
};
person.greet(); // Вызов метода объекта
```

### Зачем это нужно

Понимание типов данных в JavaScript важно для правильного использования переменных и функций. Это помогает избежать ошибок, связанных с несовместимостью типов, и позволяет эффективно управлять памятью. Например, знание того, что строки неизменяемы, помогает избежать ненужных операций копирования, а понимание объектов позволяет создавать сложные структуры данных.

### Применение

Типы данных используются везде в JavaScript: от простых операций с числами и строками до сложных манипуляций с объектами и массивами. Они являются основой для построения логики приложения и взаимодействия с пользователем.
