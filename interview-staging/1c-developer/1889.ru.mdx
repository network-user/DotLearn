# Как правильно выбрать регистр

## 1️⃣ Как кратко ответить

Выбор регистра зависит от типа данных и целей учета. Для накопления количественных и суммовых данных используйте регистры накопления. Для хранения сведений о состоянии объектов — регистры сведений. Для учета взаиморасчетов — регистры бухгалтерии. Для оперативного учета остатков и движения товаров — регистры остатков.

## 2️⃣ Подробное объяснение темы

В системе 1С регистры используются для хранения и обработки данных. Они позволяют организовать учет различной информации в зависимости от задач, которые необходимо решить. Правильный выбор регистра — это ключ к эффективной работе системы и корректному учету данных.

### Типы регистров

1. **Регистры накопления**:

**Назначение**: Используются для учета количественных и суммовых данных. Они позволяют накапливать информацию о движении и остатках товаров, денежных средств и других ресурсов.
2. **Применение**: Например, учет продаж, закупок, движения денежных средств.
3. **Пример**:```1C
РегистрНакопления.Продажи.Записать();
```
Здесь создается запись в регистре накопления "Продажи", фиксирующая факт продажи товара.
4. **Регистры сведений**:

**Назначение**: Хранят информацию о состоянии объектов на определенный момент времени. Они не предназначены для накопления данных, а скорее для хранения актуальных сведений.
5. **Применение**: Например, хранение информации о текущих ценах, курсах валют, остатках на складах.
6. **Пример**:```1C
РегистрСведений.ТекущиеЦены.Записать();
```
Здесь фиксируется текущая цена товара в регистре сведений "ТекущиеЦены".
7. **Регистры бухгалтерии**:

**Назначение**: Используются для ведения бухгалтерского учета. Они фиксируют хозяйственные операции в разрезе счетов бухгалтерского учета.
8. **Применение**: Например, учет взаиморасчетов с контрагентами, учет основных средств.
9. **Пример**:```1C
БухгалтерскийУчет.ЗаписатьПроводку();
```
Здесь создается бухгалтерская проводка, отражающая хозяйственную операцию.
10. **Регистры остатков**:

**Назначение**: Предназначены для оперативного учета остатков и движения товаров. Они позволяют быстро получать информацию о текущих остатках на складах.
11. **Применение**: Например, учет остатков товаров на складах, учет остатков материалов в производстве.
12. **Пример**:```1C
РегистрОстатков.ТоварыНаСкладах.Записать();
```
Здесь фиксируется движение товара в регистре остатков "ТоварыНаСкладах".

### Как выбрать регистр

- **Анализ данных**: Определите, какие данные необходимо учитывать и в каком разрезе. Это поможет выбрать подходящий тип регистра.
- **Цели учета**: Поймите, для каких целей будет использоваться информация. Например, для анализа продаж лучше использовать регистры накопления, а для хранения актуальных сведений — регистры сведений.
- **Частота обновления данных**: Если данные часто обновляются и важна их актуальность, выбирайте регистры сведений. Для накопления данных за период времени — регистры накопления.
- **Требования к отчетности**: Учитывайте, какие отчеты необходимо формировать и какие данные для этого нужны.

Правильный выбор регистра обеспечивает корректность учета и эффективность работы системы 1С.
