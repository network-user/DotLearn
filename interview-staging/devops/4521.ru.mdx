# Как посмотреть свободное место на диске в Linux

## 1️⃣ Как кратко ответить

Используйте команду `df -h` в терминале Linux, чтобы отобразить информацию о свободном и используемом пространстве на всех смонтированных файловых системах в удобочитаемом формате.

## 2️⃣ Подробное объяснение темы

В Linux для проверки свободного места на диске используется команда `df` (disk free). Она отображает информацию о файловых системах, включая объем доступного и используемого пространства. Команда `df` является стандартным инструментом, который можно использовать в большинстве дистрибутивов Linux.

### Основные параметры команды `df`

- `-h`: Выводит информацию в удобочитаемом формате (человеко-читаемый формат), используя единицы измерения, такие как KB, MB, GB.
- `-T`: Показывает тип файловой системы.
- `-i`: Отображает информацию об inode, что полезно для понимания, сколько файлов может быть создано на файловой системе.

### Пример использования

```bash
df -h
```

### Объяснение вывода

- **Filesystem**: Имя файловой системы или устройства.
- **Size**: Общий размер файловой системы.
- **Used**: Объем использованного пространства.
- **Avail**: Объем доступного пространства.
- **Use%**: Процент использованного пространства.
- **Mounted on**: Точка монтирования, где файловая система доступна в файловой иерархии.

### Пример вывода

```
Filesystem      Size  Used Avail Use% Mounted on
/dev/sda1        50G   20G   28G  42% /
tmpfs           1.9G     0  1.9G   0% /dev/shm
/dev/sda2       100G   60G   35G  64% /home
```

### Разбор вывода

- `/dev/sda1`: Основной раздел, смонтированный в корневую директорию `/`. Общий размер 50G, из которых 20G занято, 28G свободно, что составляет 42% использования.
- `tmpfs`: Временная файловая система, используемая для хранения временных данных. Она полностью свободна.
- `/dev/sda2`: Раздел, смонтированный в `/home`, где хранятся пользовательские данные. Из 100G занято 60G, свободно 35G, что составляет 64% использования.

### Зачем это нужно

Понимание использования дискового пространства важно для управления ресурсами системы. Это позволяет:

- Избежать переполнения диска, что может привести к сбоям в работе приложений.
- Планировать расширение дискового пространства.
- Оптимизировать использование ресурсов, удаляя ненужные файлы или перемещая данные на другие носители.

### Практическое применение

Администраторы систем регулярно используют `df` для мониторинга состояния дисков, особенно на серверах, где стабильность и производительность критически важны. Это также полезно для пользователей, которые хотят убедиться, что у них достаточно места для установки новых программ или хранения данных.
