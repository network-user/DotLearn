# Какие базовые SQL-команды

## 1️⃣ Как кратко ответить

Базовые SQL-команды включают SELECT для выборки данных, INSERT для добавления новых записей, UPDATE для изменения существующих данных, DELETE для удаления записей, CREATE для создания новых таблиц или баз данных, и DROP для удаления таблиц или баз данных.

## 2️⃣ Подробное объяснение темы

SQL (Structured Query Language) — это язык программирования, используемый для управления и манипуляции данными в реляционных базах данных. Основные команды SQL позволяют выполнять операции с данными и структурой базы данных. Рассмотрим каждую из них подробнее.

### SELECT

Команда SELECT используется для выборки данных из одной или нескольких таблиц. Она позволяет извлекать нужные данные, применяя фильтры и сортировку.

```sql
SELECT column1, column2
FROM table_name
WHERE condition;
```

- `SELECT column1, column2`: указывает, какие столбцы нужно выбрать.
- `FROM table_name`: указывает таблицу, из которой извлекаются данные.
- `WHERE condition`: фильтрует записи, возвращая только те, которые соответствуют условию.

### INSERT

Команда INSERT добавляет новые записи в таблицу.

```sql
INSERT INTO table_name (column1, column2)
VALUES (value1, value2);
```

- `INSERT INTO table_name (column1, column2)`: указывает таблицу и столбцы, в которые будут добавлены данные.
- `VALUES (value1, value2)`: задает значения для добавляемой записи.

### UPDATE

Команда UPDATE изменяет существующие данные в таблице.

```sql
UPDATE table_name
SET column1 = value1, column2 = value2
WHERE condition;
```

- `UPDATE table_name`: указывает таблицу, в которой будут обновлены данные.
- `SET column1 = value1, column2 = value2`: задает новые значения для указанных столбцов.
- `WHERE condition`: определяет, какие записи будут обновлены.

### DELETE

Команда DELETE удаляет записи из таблицы.

```sql
DELETE FROM table_name
WHERE condition;
```

- `DELETE FROM table_name`: указывает таблицу, из которой будут удалены записи.
- `WHERE condition`: определяет, какие записи будут удалены. Без условия удаляются все записи.

### CREATE

Команда CREATE используется для создания новых таблиц или баз данных.

```sql
CREATE TABLE table_name (
    column1 datatype,
    column2 datatype
);
```

- `CREATE TABLE table_name`: создает новую таблицу с указанным именем.
- `column1 datatype, column2 datatype`: определяет столбцы и их типы данных.

### DROP

Команда DROP удаляет существующие таблицы или базы данных.

```sql
DROP TABLE table_name;
```

- `DROP TABLE table_name`: удаляет таблицу с указанным именем из базы данных.

Эти команды являются основой для работы с реляционными базами данных и позволяют выполнять основные операции по управлению данными и структурой базы данных.
