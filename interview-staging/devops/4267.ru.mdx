# Какой командой посмотреть IP-адрес на Linux-сервере

## 1️⃣ Как кратко ответить

Команда `ip addr show` позволяет отобразить IP-адреса всех сетевых интерфейсов на Linux-сервере.

## 2️⃣ Подробное объяснение темы

Для управления сетевыми интерфейсами и получения информации о них в Linux используется утилита `ip`, которая является частью пакета `iproute2`. Эта утилита предоставляет более современный и мощный способ управления сетевыми настройками по сравнению с устаревшей командой `ifconfig`.

Чтобы узнать IP-адреса всех сетевых интерфейсов на Linux-сервере, используется команда:

```bash
ip addr show
```

### Разбор команды

- `ip`: это основная утилита для управления сетевыми интерфейсами, маршрутизацией и туннелями.
- `addr`: подкоманда, которая используется для работы с адресами интерфейсов.
- `show`: действие, которое указывает утилите вывести текущую информацию.

### Пример вывода

```plaintext
1: lo: <LOOPBACK,UP,LOWER_UP> mtu 65536 qdisc noqueue state UNKNOWN group default qlen 1000
    link/loopback 00:00:00:00:00:00 brd 00:00:00:00:00:00
    inet 127.0.0.1/8 scope host lo
       valid_lft forever preferred_lft forever
    inet6 ::1/128 scope host 
       valid_lft forever preferred_lft forever
2: eth0: <BROADCAST,MULTICAST,UP,LOWER_UP> mtu 1500 qdisc pfifo_fast state UP group default qlen 1000
    link/ether 02:42:ac:11:00:02 brd ff:ff:ff:ff:ff:ff
    inet 172.17.0.2/16 brd 172.17.255.255 scope global eth0
       valid_lft forever preferred_lft forever
    inet6 fe80::42:acff:fe11:2/64 scope link 
       valid_lft forever preferred_lft forever
```

### Объяснение вывода

- **lo**: это интерфейс loopback, который используется для связи внутри самого устройства. Его IP-адрес всегда `127.0.0.1`.
- **eth0**: это пример физического или виртуального сетевого интерфейса. В данном случае, его IP-адрес — `172.17.0.2`.

### Зачем это нужно

Знание IP-адреса необходимо для настройки сетевых соединений, диагностики сетевых проблем и обеспечения безопасности. Например, при настройке брандмауэра или VPN-соединения важно знать, какие IP-адреса используются сервером.

### Применение

Эта команда полезна для системных администраторов и специалистов DevOps, которые управляют сетевой инфраструктурой, настраивают серверы и обеспечивают их бесперебойную работу. Она позволяет быстро получить информацию о сетевых интерфейсах и их конфигурации, что важно для диагностики и устранения неполадок.
